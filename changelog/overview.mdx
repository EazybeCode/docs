# Changelog

Latest updates and improvements to Eazybe

<Update label="January 11, 2026">
### Added - AI Unreplied Chats Agent

We've introduced intelligent chat prioritization to help your team focus on what matters most.

- Automatic Critical/Important/Normal classification based on keywords and sentiment
- WhatsApp notifications for admins every 3 hours with summary stats
- Smart detection of urgent keywords like "cancel", "refund", "urgent"
- CRM-based customer value recognition for VIP prioritization

### Added - Enhanced Privacy Controls

New privacy and compliance features to keep your data secure:

- Complete Privacy & Compliance documentation
- GDPR compliance tools with DPA support
- Employee monitoring transparency features
- Label-based chat exclusion from backup
</Update>

<Update label="January 5, 2026">
### Added - AI Reply Suggestions

Get AI-powered message suggestions to respond faster and more professionally.

- Multiple tone options: Professional, Friendly, Concise, Persuasive, Empathetic
- One-click message insertion
- Works in any WhatsApp conversation
- No message data stored after processing

### Fixed - HubSpot Sync Improvements

- Fixed contact matching for numbers without country code
- Improved attachment sync reliability
- Reduced sync delay from 5 min to 3 min average
</Update>
