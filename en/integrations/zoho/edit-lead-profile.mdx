## Overview

This guide explains how to edit and modify lead information within the Eazybe WhatsApp Extension when connected to your Zoho CRM leads.

## How to Edit a Lead Profile

<Steps>
  <Step title="Open the WhatsApp Chat">
    Open the WhatsApp chat of the respective Zoho Lead whose profile you want to edit.

    ![Open WhatsApp Chat](/images/1223641553.png)
  </Step>

  <Step title="Click on the Edit Button">
    Click on the **Edit** button in the Contact Card of the Profile Section that appears in the Mini-CRM panel.
  </Step>

  <Step title="Edit Lead Details">
    Edit or select user properties as required:
    - Modify existing field values
    - Update contact information
    - Specify whether each selected property is required or optional

    ![Edit Lead Details](/images/1223641691.png)
  </Step>

  <Step title="Save the Changes">
    Click on **Apply** to update the Contact Card with the properties that you have selected.

    Edit the properties fields as needed and click on **Save** to finalize your changes.
  </Step>
</Steps>

<Tip>
  Remember to save your changes before navigating away to ensure all updates are properly recorded in Zoho CRM.
</Tip>

## Editable Fields

You can typically edit the following lead properties:

| Field | Description |
|-------|-------------|
| **Name** | Lead's full name |
| **Email** | Primary email address |
| **Phone** | Contact phone number |
| **Company** | Associated company name |
| **Lead Status** | Current status in pipeline |
| **Lead Source** | How the lead was acquired |
| **Custom Fields** | Any custom fields configured in Zoho |
