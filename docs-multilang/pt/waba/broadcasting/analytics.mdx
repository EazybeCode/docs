## Campaign Metrics

After sending a broadcast, track these key metrics:

| Metric | Description | How Calculated |
|--------|-------------|----------------|
| **Sent** | Messages submitted to WhatsApp | Total contacts attempted |
| **Delivered** | Messages reached recipient's phone | ✓✓ double check mark |
| **Read** | Messages opened by recipient | Blue ✓✓ marks |
| **Failed** | Messages that couldn't be delivered | Sent - Delivered |
| **Response Rate** | Recipients who replied | Replies ÷ Delivered |

## Viewing Analytics

### Campaign Level

1. Go to **Broadcasts** in the menu
2. Find your campaign in the list
3. Click to view detailed metrics

### Message Level

Click on individual contacts to see:
- Delivery timestamp
- Read timestamp
- Any replies received

## Performance Benchmarks

| Metric | Good | Excellent |
|--------|------|-----------|
| **Delivery Rate** | > 90% | > 98% |
| **Read Rate** | > 50% | > 70% |
| **Response Rate** | > 5% | > 15% |

## Common Failure Reasons

| Reason | Solution |
|--------|----------|
| **Invalid number** | Verify phone format (include country code) |
| **Not on WhatsApp** | Remove non-WhatsApp numbers from list |
| **Blocked** | Respect opt-out requests |
| **Rate limited** | Reduce sending speed |
| **Template rejected** | Use an approved template |
