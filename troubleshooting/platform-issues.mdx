## Overview

This guide helps you resolve common issues with the Eazybe platform.

## Connection Issues

### WhatsApp Not Connecting

<Steps>
  <Step title="Check WhatsApp Phone">
    Ensure your phone has internet connection
  </Step>
  <Step title="Refresh Browser">
    Hard refresh with Ctrl+Shift+R (Cmd+Shift+R on Mac)
  </Step>
  <Step title="Re-scan QR Code">
    Log out of WhatsApp Web and scan again
  </Step>
</Steps>

### Eazybe Not Loading

1. Clear browser cache and cookies
2. Disable other browser add-ons temporarily
3. Try in incognito/private mode
4. Check if eazybe.com is accessible

## Sync Issues

### Chats Not Syncing

- Ensure Google Drive is connected
- Check if sync is paused in settings
- Verify internet connection is stable

### CRM Data Not Showing

- Verify CRM integration is active
- Re-authorize if token expired
- Check field mapping settings

## Getting Help

If issues persist:

1. **Check Status**: [status.eazybe.com](https://status.eazybe.com)
2. **Contact Support**: [support@eazybe.com](mailto:support@eazybe.com)
