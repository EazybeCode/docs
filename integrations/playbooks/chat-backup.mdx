## Overview

Never lose a customer conversation again. This playbook shows you how to automatically backup every WhatsApp message to your CRM contact timeline.

<Info>
**Setup Time**: 5 minutes  
**Difficulty**: Easy  
**Works With**: HubSpot, Zoho, Salesforce, Pipedrive, Google Sheets
</Info>

## The Problem

85-90% of sales conversations happen on WhatsApp, but:
- Messages live only on the rep's phone
- When reps leave, conversations disappear
- Managers have zero visibility
- Handoffs between reps lose context

## Step-by-Step Setup

<Steps>
  <Step title="Enable Chat Backup">
    Go to **Settings** → **CRM Integration** → Enable **Chat Backup**
    
    <Frame>
      <img src="/images/playbooks/enable-chat-backup.png" alt="Enable chat backup toggle" />
    </Frame>
  </Step>
  
  <Step title="Choose Backup Destination">
    Select where messages should be logged:
    
    | Option | Best For |
    |--------|----------|
    | **Contact Timeline** | See full conversation history |
    | **Notes** | Quick reference |
    | **Custom Activity** | Advanced reporting |
    
    <Frame>
      <img src="/images/playbooks/backup-destination.png" alt="Backup destination options" />
    </Frame>
  </Step>
  
  <Step title="Configure Sync Frequency">
    Choose how often to sync:
    - **Real-time** - Messages sync instantly (recommended)
    - **Every 5 minutes** - Batch sync
    - **Manual** - Sync on demand
  </Step>
  
  <Step title="Select What to Backup">
    Choose which content to include:
    - ✅ Text messages
    - ✅ Images & attachments
    - ✅ Voice notes (as links)
    - ✅ Documents
    - ⬜ Stickers & GIFs (optional)
  </Step>
</Steps>

## Best Practices

<AccordionGroup>
  <Accordion title="Enable for all team members">
    Ensure every sales rep has chat backup enabled. Go to **Settings** → **Team** to verify.
  </Accordion>
  
  <Accordion title="Use real-time sync">
    Real-time sync ensures no messages are lost if a rep disconnects or changes devices.
  </Accordion>
  
  <Accordion title="Include attachments">
    Images and documents often contain important information (quotes, contracts, product photos).
  </Accordion>
</AccordionGroup>

## Privacy Considerations

<Warning>
Chat backup captures all messages including personal chats if using a personal WhatsApp number. Consider:
- Using WhatsApp Business accounts
- Informing team members about backup
- Enabling Privacy Blur for sensitive content
</Warning>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Messages not appearing in CRM?">
    1. Check sync status in Eazybe sidebar
    2. Verify CRM connection is active
    3. Ensure contact exists in CRM (enable auto-create)
  </Accordion>
  
  <Accordion title="Attachments not syncing?">
    Some CRM plans have attachment limits. Check your CRM's file storage quota.
  </Accordion>
  
  <Accordion title="Delayed sync?">
    Switch to real-time sync if using batch mode. Check internet connection.
  </Accordion>
</AccordionGroup>
