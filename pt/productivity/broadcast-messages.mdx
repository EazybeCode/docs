Quer enviar uma mensagem para múltiplos contatos ao mesmo tempo? O recurso de mensagens de transmissão do Eazybe facilita alcançar toda a sua audiência com uma única ação.

## Como Enviar Mensagens de Transmissão

<Steps>
  <Step title="Abrir Mensagens de Transmissão">
    Clique no botão **Mensagens de Transmissão** na barra superior.

    ![Botão de Mensagens de Transmissão](/images/1112608994.png)
  </Step>
  <Step title="Selecionar uma Resposta Rápida">
    Selecione a Resposta Rápida do menu suspenso que você deseja enviar como transmissão.

    ![Selecionar Resposta Rápida](/images/1112609104.png)
  </Step>
  <Step title="Escolher Rótulo de Destino">
    Selecione o Rótulo para o qual você deseja enviar a resposta rápida. Isso permite que você direcione grupos específicos de contatos.

    ![Selecionar Rótulo](/images/1112609231.png)
  </Step>
  <Step title="Enviar ou Criar Nova">
    Você tem duas opções:
    - Clique em **Enviar Mensagem** para enviar a resposta rápida selecionada
    - Clique em **Criar Nova Resposta Rápida** para criar uma nova mensagem para esta transmissão

    ![Enviar Mensagem](/images/1112609389.png)
  </Step>
</Steps>

<Warning>
Este recurso deve ser usado com cautela e de forma limitada. Certifique-se de ler o aviso de isenção de responsabilidade cuidadosamente após clicar no botão Enviar Mensagem.
</Warning>

<Tip>
Organize seus contatos com Rótulos antes de usar mensagens de transmissão para garantir que sua mensagem alcance o público certo.
</Tip>
