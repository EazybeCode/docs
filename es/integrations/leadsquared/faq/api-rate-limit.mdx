Si estás viendo una notificación sobre el límite de API excedido, este artículo explica los límites de tasa vigentes al trabajar con la integración de LeadSquared.

## Información sobre el Límite de Tasa

<Warning>
  Existe un límite de tasa de **5 llamadas de API por segundo** al trabajar con la integración de LeadSquared.
</Warning>

El sistema EazyBe ha sido optimizado para manejar esta restricción de manera eficiente. Sin embargo, cuando recibes una notificación sobre límites de API, indica que se ha excedido el umbral de 5 llamadas por segundo.

## Por Qué Sucede Esto

La notificación de límite de tasa de API puede aparecer cuando:

- Múltiples conversaciones de WhatsApp se están sincronizando simultáneamente
- Se realizan acciones rápidas en el panel Mini-CRM
- Varios miembros del equipo están usando activamente la integración al mismo tiempo

<Info>
  Esta es una limitación impuesta por la API de LeadSquared, no por EazyBe. El límite de tasa ayuda a garantizar un servicio estable para todos los usuarios.
</Info>

## Qué Hacer

<Tip>
  Si encuentras frecuentemente la notificación de límite de API:
  - Espera unos segundos antes de reintentar tu acción
  - Espacía tus actividades al realizar múltiples operaciones
  - El sistema reintentará automáticamente las solicitudes fallidas cuando se reinicie el límite
</Tip>
