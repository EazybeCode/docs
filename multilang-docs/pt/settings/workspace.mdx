O espaço de trabalho EazyBe foi projetado especialmente para você gerenciar o perfil da sua organização de forma integrada e fácil.

**Como acessar o espaço de trabalho do EazyBe?**

- Clique no botão de organização na barra lateral direita conforme mostrado abaixo.
- Uma nova guia será aberta e faça login usando as credenciais de conta desejadas.
- Você verá a interface do EazyBe Workspace.
