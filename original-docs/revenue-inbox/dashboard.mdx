## Conversation List

The main area shows all conversations matching your filters.

### Conversation Card Elements

Each conversation card displays:

| Element | Description |
|---------|-------------|
| Contact Name | WhatsApp name or CRM-linked name |
| Last Message | Preview of most recent message |
| Timestamp | When last message was sent/received |
| Rep Name | Team member handling this conversation |
| Labels | Applied labels (Hot Lead, Follow-up, etc.) |
| Status Icon | Unread, unreplied, or normal |
| CRM Link | Icon showing CRM connection status |

### Sorting Options

Sort conversations by:
- **Most Recent** - Newest activity first
- **Oldest First** - Oldest messages first
- **Unreplied Duration** - Longest wait time first
- **Contact Name** - Alphabetical
