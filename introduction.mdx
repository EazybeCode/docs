## How Eazybe Works

Eazybe consists of multiple components that work together:

| Component | What It Does |
|-----------|--------------|
| **Chrome Extension** | Enhances WhatsApp Web with CRM sidebar, labels, quick replies |
| **Revenue Inbox** | Web dashboard for team collaboration and analytics |
| **WABA Integration** | WhatsApp Business API for broadcasting and chatbots |
| **CRM Connectors** | Two-way sync with HubSpot, Salesforce, Zoho, etc. |

<Info>
The Chrome extension works alongside WhatsApp Web to add powerful features without changing how you use WhatsApp.
</Info>
