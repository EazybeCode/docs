Do you have an important conversation that you don't want to lose track of, going on, or a discussion that paused mid-way before reaching a conclusion? Don't worry! With the Notes section in EazyBe LeadSquared Mini-CRM view, you can create follow-up notes and save key messages for future reference

But you might be wondering, how?

Letâ€™s find out.

- 
#### Select the Contact
Open the contact for which you want to create a follow-up. Ensure the contact exist on LeadSquared.

- 
#### Access the Notes Section
Click on the LeadSquared icon. The second section in the top bar will take you to the **Notes section**.

- 
#### Add a New Note
Click on the **Add Note** button to add notes for the respective lead.

  

- 
#### Enter Note Details
Enter the note or description you want to store for the conversation, then click **Save**. The note will be saved along with the time of creation.

  

- 
#### Edit or Delete Notes
You can **Edit** or **Delete** a note, click on the three dots at the top and select the desired action.

- 
#### Add Multiple Notes
You can add additional notes by clicking on the **Add Note** button and it will create new notes for that lead.

  

Note: Create Notes for LeadSquared OpportunitiesYou can also create **Notes** for any LeadSquared Opportunity in a similar way as tasks:

- Click on the respective opportunity for which you want to add a note.
- Follow the same steps to add a note by clicking on the **Add Note** button.
- Once the note is added, it will be saved under that specific opportunity for future reference.
