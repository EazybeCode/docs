## Dashboard

The Revenue Inbox Dashboard provides a quick overview of your team's WhatsApp performance.

### Key Metrics

| Metric | Description |
|--------|-------------|
| **Total Conversations** | All chats in the selected period |
| **Unreplied Chats** | Awaiting team response |
| **Critical Chats** | AI-flagged urgent conversations |
| **Avg Response Time** | Time to first reply |
| **Resolution Rate** | % of chats resolved |

### Accessing the Dashboard

1. Go to Eazybe Workspace
2. Click **Dashboard** in the left sidebar
3. Select date range for analysis
4. View metrics and charts

<Info>
Export dashboard data to CSV for reporting. Click **Export** in the top right.
</Info>
