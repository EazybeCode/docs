## How to Send quick reply

1.Click on the Quick Reply button.

  

- Give a name to the Quick Reply and type in the message you want to add in the same.

  

- You have an option to create a Quick Reply for Personal - only for yourself or Shared - for the entire organization.

  

- You have a checkbox to Edit the reply before sending. You can select it as per your needs.

  

- There is also an option to rephrase the message using AI or to record an audio or send an attachment with a quick reply.

  

- Once you have entered the details, click on the Create button and your quick reply will be created.

**How do I set the visibility of a Quick Reply?**

*When creating a Quick Reply in Step 2, you can choose the visibility options: "Personal" (Visible only to you), "Shared" (Visible to your entire Organization), or "Shared with team" (Visible only to specific teams within the Organization).*

**Can I add attachments or voice notes to a Quick Reply?**

*Yes, you have the option to add an attachment or voice note to a Quick Reply. Note that the file size limit for attachments is 20MB*

**Are there any restrictions on file types for attachments in Quick Replies?**

*Eazybe supports various file types for attachments in Quick Replies. As of now, the file size limit is 20MB, and only one file can be attached.*

**Is there a limit to the number of Quick Replies I can create?**

*There is no limit to the number of Quick Replies you can create. You can create as many as needed for different scenarios and messages.*

**Can I edit or delete Quick Replies after creating them?**

*Yes, you can edit or delete Quick Replies as needed. The interface allows you to manage and make changes to your Quick Replies.*
