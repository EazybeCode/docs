## Overview

This guide provides step-by-step instructions on how to link your WABA API numbers to specific HubSpot users. Successfully mapping users ensures accurate data synchronization and efficient workflow management between your platforms.

<Note>
Before proceeding, ensure you have **Admin permissions** in both Eazybe and HubSpot to manage user mappings.
</Note>

---

## Step-by-Step Guide

<Steps>
  <Step title="Navigate to the Integrations Page">
    Go to the Eazybe dashboard and navigate to the **Integrations** section.

    ![Navigate to Integrations Page](/images/waba-hubspot-1.png)

    <Tip>
    You can access the Integrations page from the main menu on the left sidebar.
    </Tip>
  </Step>

  <Step title="Access HubSpot Integration Settings">
    Find the **HubSpot** integration card and click on it to access the integration settings.

    ![Access HubSpot Integration Settings](/images/waba-hubspot-2.png)
  </Step>

  <Step title="Select the User Mapping Tab">
    In the HubSpot integration settings, locate and click on the **User Mapping** tab.

    ![Select User Mapping Tab](/images/waba-hubspot-3.png)

    <Info>
    This tab allows you to assign specific HubSpot users to manage your WABA API numbers.
    </Info>
  </Step>

  <Step title="Choose a HubSpot User to Map">
    Select the **HubSpot user** you want to map to a specific WABA API number from the available user list.

    ![Choose HubSpot User to Map](/images/waba-hubspot-4.png)
  </Step>

  <Step title="Select the Specific HubSpot User">
    Choose the specific HubSpot user from the dropdown menu that you wish to associate with your WABA API number.

    ![Select Specific HubSpot User](/images/waba-hubspot-5.png)

    <Warning>
    Ensure the selected user has the necessary permissions in HubSpot to manage API communications.
    </Warning>
  </Step>

  <Step title="Verify Successful User Mapping">
    Confirm the mapping and verify that the user has been successfully linked to the WABA API number.

    ![Verify Successful User Mapping](/images/waba-hubspot-6.png)

    <Tip>
    You can modify or remove user mappings at any time from the User Mapping tab.
    </Tip>
  </Step>
</Steps>

## Troubleshooting

<Accordion>
  <AccordionItem title="User not appearing in the list?">
    Ensure the user has been invited to your HubSpot portal and has accepted the invitation. Only active HubSpot users with proper permissions will appear in the mapping list.
  </AccordionItem>
  <AccordionItem title="Mapping not saving?">
    Check that you have the required admin permissions. If the issue persists, try refreshing the page and attempting the mapping again.
  </AccordionItem>
  <AccordionItem title="Can I map multiple API numbers to one user?">
    Yes, a single HubSpot user can be mapped to multiple WABA API numbers. This is useful for team members managing multiple business numbers.
  </AccordionItem>
</Accordion>
