title: "Baglanti Kesme Kontrolu"
description: "EazyBe Cloud WhatsApp baglanti kesme isleminizin basarili olup olmadigini nasil dogrulayacaginizi ogrenin"
---

Cloud WhatsApp'inizin basariyla baglantisininin kesildigini dogrulamanin iki yolu vardir:

## Yontem 1: Cloud Simgesini Kontrol Edin

EazyBe uzantisindaki bulut simgesi, baglanti kesme basarili oldugunda **kirmizi** olacaktir.

<Tip>
Kirmizi bulut simgesi, Cloud WhatsApp'inizin artik bagli olmadigini dogrular.
</Tip>

---

## Yontem 2: Workspace'inizi Kontrol Edin

<Steps>
  <Step title="EazyBe Workspace'i Acin">
    EazyBe Workspace panelinize gidin.
  </Step>

  <Step title="Durum Gostergesini Kontrol Edin">
    Durum gostergesi **Stopped** (Durduruldu) gosterecektir.
  </Step>

  <Step title="Buton Metnini Dogrulayin">
    Cloud Baglanti butonu **Restart WhatsApp** gosterecektir.

    ![Workspace'te Baglanti Kesme Durumu](/images/1203770785.png)
  </Step>
</Steps>
