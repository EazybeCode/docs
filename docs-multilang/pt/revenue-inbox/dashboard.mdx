## Overview

The Team Monitoring Dashboard gives managers complete visibility into their team's WhatsApp activity without needing access to individual phones. Track response times, identify bottlenecks, and ensure no customer conversation falls through the cracks.

## Dashboard Metrics

### Employee Overview

| Column | Description |
|--------|-------------|
| **Name** | Team member name |
| **Last Activity** | When they were last active on WhatsApp Web |
| **Last Chat Synced** | When their most recent chat was backed up |
| **Total Chats** | Number of conversations in backup |
| **Unreplied** | Count of chats waiting for response |
| **CRM Status** | Integration connection status |

### Accessing the Dashboard

<Steps>
  <Step title="Open Workspace">
    Log into **Eazybe Workspace** with admin credentials
  </Step>
  
  <Step title="Navigate to Dashboard">
    Click **Dashboard** in the left sidebar
  </Step>
  
  <Step title="View Team Data">
    See all team members with their activity status, sync times, and integrations
  </Step>
  
  <Step title="Drill Down">
    Click on any employee to view their individual conversations and performance
  </Step>
</Steps>

## Analytics Tab

The Analytics tab provides detailed message analytics for your entire team or individual employees:

<CardGroup cols={3}>
  <Card title="Messages Sent" icon="paper-plane">
    Total outbound messages by team/individual
  </Card>
  <Card title="Messages Received" icon="inbox">
    Total inbound customer messages
  </Card>
  <Card title="Response Rate" icon="reply">
    Percentage of messages replied to
  </Card>
</CardGroup>

### Time-Based Analytics

| Metric | What It Shows |
|--------|---------------|
| **Daily Volume** | Messages sent/received per day |
| **Peak Hours** | Busiest times for customer messages |
| **Response Time** | Average time to first reply |
| **Resolution Time** | Average time to conversation completion |

### Filtering Options

- **All Employees** or select specific team member
- **All Teams** or filter by team/department
- **Unreplied** vs **All Conversations**
- **Critical** priority filter

## Monitoring Best Practices

<AccordionGroup>
  <Accordion title="Daily Check-ins" icon="sun">
    Review the Dashboard each morning to:
    - Check overnight unreplied chats
    - Identify any sync issues
    - Spot team members with high unreplied counts
  </Accordion>
  
  <Accordion title="Weekly Reviews" icon="calendar-week">
    Analyze weekly trends:
    - Compare response times across team
    - Identify top performers
    - Find coaching opportunities
  </Accordion>
  
  <Accordion title="Real-time Alerts" icon="bell">
    Set up notifications for:
    - Critical unreplied chats
    - Sync failures
    - SLA breaches
  </Accordion>
</AccordionGroup>
