Eazybe é uma extensão do Chrome que aprimora o WhatsApp Web com recursos de produtividade e integração com CRM para equipes de vendas e suporte. Os principais recursos incluem mini-visualizações de CRM, caixas de entrada de equipe compartilhadas, etiquetas de contatos e sincronização de dados para Google Drive ou sistemas CRM.

## Tratamento e Armazenamento de Dados

| Categoria | Detalhes |
|-----------|----------|
| **Armazenamento de Mensagens** | O Eazybe não armazena mensagens. Todos os chats permanecem no WhatsApp. |
| **Dados de CRM** | Exibidos via APIs; credenciais não armazenadas; OAuth preferido |
| **Backups** | Armazenados no Google Drive do usuário |
| **Anexos** | Mídia armazenada no Google Drive quando o backup está habilitado |
| **Armazenamento em Servidor** | Os dados permanecem do lado do cliente ou nos serviços de nuvem do usuário |

## Padrões de Backup de Chat

<Warning>
Quando o Team Inbox está habilitado, todos os chats do WhatsApp são backupeados por padrão no Google Drive.
</Warning>

Os usuários podem controlar as configurações de backup através de:
- **Etiquetas "Não Sincronizar"** para contatos específicos
- **Controles de administrador** para regras de sincronização de toda a organização

## Conectividade Cloud WhatsApp

Recurso opcional que permite mensagens agendadas/automatizadas quando o navegador está fechado:

- Nenhum dado do WhatsApp armazenado nos servidores do Eazybe
- Autenticação via código QR do WhatsApp
- Sessões podem ser desconectadas a qualquer momento pela extensão

## Permissões Necessárias

| Permissão | Propósito |
|-----------|-----------|
| **Acesso às Abas do Chrome** | Sobreposição de UI para WhatsApp Web |
| **Acesso ao CRM** | Integração OAuth/API com HubSpot, Zoho, Salesforce, etc. |
| **Google Drive** | Backup opcional e armazenamento do Team Inbox |

## Requisitos de Configuração de TI

<Steps>
  <Step title="Requisitos do Navegador">
    Navegador Google Chrome necessário com login no WhatsApp Web no mesmo perfil do Chrome.
  </Step>
  <Step title="Permissões da Extensão">
    Habilitar as permissões da extensão Chrome para o Eazybe.
  </Step>
  <Step title="Opcional: Integração de CRM">
    Configurar chaves de API do CRM ou credenciais OAuth se usar recursos de CRM.
  </Step>
  <Step title="Opcional: Google Drive">
    Conceder acesso ao Google Drive para administradores do workspace se usar recursos de backup.
  </Step>
  <Step title="Configuração de Firewall">
    Permitir HTTPS para `web.eazybe.com`, `api.eazybe.com` e seus endpoints de CRM.
  </Step>
</Steps>

<Tip>
O Eazybe não requer instalação de software nas máquinas dos usuários - apenas uma extensão leve do Chrome. Recomendamos usar contas de teste para avaliação segura.
</Tip>
