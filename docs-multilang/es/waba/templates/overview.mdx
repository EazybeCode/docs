## What Are Message Templates?

**Message templates** are pre-approved message formats required by WhatsApp for:

- Starting conversations with customers (outside 24-hour window)
- Sending promotional/marketing messages
- Broadcasts to multiple recipients

Templates ensure businesses send high-quality, non-spammy messages to users.

## Template Components

Every template can include:

| Component | Description | Required |
|-----------|-------------|----------|
| **Header** | Image, video, document, or text | Optional |
| **Body** | Main message content with variables | Required |
| **Footer** | Small text (e.g., "Reply STOP to unsubscribe") | Optional |
| **Buttons** | Call-to-action or quick reply buttons | Optional |

### Variables

Use placeholders for personalization:

```
Hello {{1}}, your order #{{2}} has been shipped!
Track it here: {{3}}
```

## Template Lifecycle

<Steps>
  <Step title="Create">
    Design your template in Eazybe Workspace
  </Step>
  
  <Step title="Submit">
    Template is sent to Meta for review
  </Step>
  
  <Step title="Review">
    Meta reviews (typically 24-48 hours)
  </Step>
  
  <Step title="Approved / Rejected">
    If approved: Ready to use
    
    If rejected: Edit and resubmit
  </Step>
  
  <Step title="Use">
    Select template when creating broadcasts
  </Step>
</Steps>

## Template Status

| Status | Meaning |
|--------|---------|
| **Pending** | Under Meta review |
| **Approved** | Ready to use |
| **Rejected** | Failed review (see reason, edit, resubmit) |
| **Paused** | Temporarily disabled (quality issues) |
| **Disabled** | Permanently blocked |
