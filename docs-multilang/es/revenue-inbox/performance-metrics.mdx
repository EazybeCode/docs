## Team Analytics Dashboard

Access comprehensive analytics at **Workspace** → **Analytics**

## Key Metrics

### Volume Metrics

| Metric | Description |
|--------|-------------|
| **Total Conversations** | All chats in the selected period |
| **Messages Sent** | Outbound messages from team |
| **Messages Received** | Inbound customer messages |
| **New Contacts** | First-time conversations |

### Performance Metrics

| Metric | Description | Target |
|--------|-------------|--------|
| **Response Rate** | % of chats responded | > 95% |
| **Avg Response Time** | Time to first reply | < 30 min |
| **Resolution Rate** | % of issues resolved | > 85% |
| **SLA Compliance** | Within target response time | > 90% |

### Engagement Metrics

| Metric | Description |
|--------|-------------|
| **Conversation Length** | Avg messages per chat |
| **Customer Satisfaction** | Feedback scores (if collected) |
| **Repeat Contact Rate** | % of returning customers |

## Individual Performance

Click on any team member to see their personal metrics:

- Their message volume
- Their response times
- Their open conversations
- Their label distribution

## Leaderboards

Compare team members across:
- Fastest average response time
- Most conversations handled
- Highest resolution rate
- Best SLA compliance

## Exporting Reports

1. Go to **Analytics**
2. Set date range and filters
3. Click **Export** → **CSV** or **PDF**
4. Use for performance reviews or reporting
