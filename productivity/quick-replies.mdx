## Overview

Quick Replies let you save frequently used messages and send them with one click. Perfect for greetings, FAQs, pricing information, and follow-ups.

## Creating a Quick Reply

<Steps>
  <Step title="Open Quick Replies">
    Click the **Quick Replies** icon in the Eazybe sidebar (lightning bolt icon).
  </Step>
  
  <Step title="Click Create">
    Click the **+ Create Quick Reply** button.
  </Step>
  
  <Step title="Enter Details">
    - **Title**: Name for easy identification (e.g., "Welcome Message")
    - **Message**: The actual text to send
    - **Shortcut** (optional): Type `/shortcut` to trigger
  </Step>
  
  <Step title="Set Visibility">
    Choose who can see this Quick Reply:
    - **Personal**: Only visible to you
    - **Shared**: Visible to entire organization
    - **Shared with Team**: Visible to specific teams only
  </Step>
  
  <Step title="Add Extras (Optional)">
    - **Attachment**: Add a file (max 20MB)
    - **Voice Note**: Record an audio message
    - **AI Rephrase**: Use AI to improve the message
  </Step>
  
  <Step title="Save">
    Click **Create** to save your Quick Reply.
  </Step>
</Steps>

## Using Quick Replies

### Method 1: Click to Send
1. Open any WhatsApp chat
2. Click the **Quick Replies** icon
3. Find your reply and click to insert
4. Edit if needed, then send

### Method 2: Shortcut Command
1. In the chat input, type `/` followed by your shortcut
2. Example: `/welcome` inserts your welcome message
3. Edit if needed, then send

### Method 3: Search
1. Open Quick Replies panel
2. Use the search bar to find by title or content
3. Click to insert

## Organizing Quick Replies

### Categories
Group related Quick Replies into categories:
- Sales (pricing, proposals, follow-ups)
- Support (troubleshooting, FAQs)
- Onboarding (welcome, next steps)
- Closing (thank you, feedback requests)

### Best Practices

<Tip>
  **Naming Convention**: Use prefixes like `sales-`, `support-`, `onboard-` to keep replies organized and easy to search.
</Tip>

## Adding Attachments

Quick Replies can include files:

| File Type | Supported | Max Size |
|-----------|-----------|----------|
| Images | ✅ | 20 MB |
| PDFs | ✅ | 20 MB |
| Documents | ✅ | 20 MB |
| Videos | ✅ | 20 MB |
| Voice Notes | ✅ | 20 MB |

<Warning>
  Only **one file** can be attached per Quick Reply.
</Warning>

## AI-Powered Rephrasing

Improve your messages with AI:

1. Write your base message
2. Click **Rephrase with AI**
3. Choose from suggested variations:
   - More professional
   - More friendly
   - More concise
   - Different tone

## Sharing Quick Replies

### Organization-Wide
Set visibility to **Shared** to make replies available to your entire team. Great for:
- Standard company responses
- Pricing information
- Policy explanations

### Team-Specific
Set visibility to **Shared with Team** for department-specific replies:
- Sales team only
- Support team only
- Managers only

## Managing Quick Replies

### Editing
1. Open Quick Replies panel
2. Find the reply to edit
3. Click the **Edit** (pencil) icon
4. Make changes and save

### Deleting
1. Open Quick Replies panel
2. Find the reply to delete
3. Click the **Delete** (trash) icon
4. Confirm deletion

<Warning>
  Deleting a shared Quick Reply removes it for all team members.
</Warning>

## FAQ

<AccordionGroup>
  <Accordion title="Is there a limit to how many Quick Replies I can create?">
    No, you can create unlimited Quick Replies.
  </Accordion>
  
  <Accordion title="Can I use variables in Quick Replies?">
    Currently, Quick Replies are static text. For personalized messages with variables, use WABA Message Templates.
  </Accordion>
  
  <Accordion title="Do Quick Replies work with WABA?">
    Quick Replies work with regular WhatsApp. For WABA (API) messaging, you need approved Message Templates.
  </Accordion>
  
  <Accordion title="Can I import Quick Replies from another system?">
    Currently, Quick Replies must be created manually. We're working on import functionality.
  </Accordion>
</AccordionGroup>
