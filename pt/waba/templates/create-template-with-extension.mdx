Criar modelos do WhatsApp diretamente da extensão Eazybe facilita o envio de mensagens aprovadas ou respostas rápidas para seus contatos sem sair do WhatsApp Web.

## Visão Geral

A extensão do navegador Eazybe permite criar modelos de mensagens instantaneamente enquanto conversa com os clientes. Isso é ideal para configurar rapidamente mensagens reutilizáveis para respostas comuns.

<Info>
Os modelos criados através da extensão são enviados ao Meta para aprovação e ficam disponíveis em todas as suas conversas após a aprovação.
</Info>

## Como Criar Modelos pela Extensão

<Steps>
  <Step title="Acessar Resposta Rápida no Campo de Chat">
    Abra a extensão Eazybe no WhatsApp Web e localize a seção de entrada de chat em qualquer conversa. Clique na opção **Resposta Rápida** abaixo do campo de entrada.

    ![Acesso à Resposta Rápida](/images/1710847024.png)
  </Step>

  <Step title="Ativar o Toggle de Criação de Modelo">
    No menu de resposta rápida, ative o toggle **"Criar Modelo"**. Isso ativa o modo de criação de modelos de mensagens reutilizáveis.

    ![Toggle Criar Modelo](/images/1710847407.png)
  </Step>

  <Step title="Preencher os Detalhes do Modelo">
    Complete os campos obrigatórios para seu modelo:

    ![Formulário de Detalhes do Modelo](/images/1726023464.png)

    | Campo | Descrição |
    |-------|-----------|
    | **Nome do Modelo** | Um nome único para seu modelo |
    | **Idioma** | Selecione o idioma do modelo |
    | **Categoria** | Escolha Marketing, Utilidade ou Autenticação |
    | **Tipo de Modelo** | Selecione o tipo apropriado |
    | **Conteúdo da Mensagem** | Escreva a mensagem do seu modelo |
    | **Rodapé** | Texto opcional no rodapé |
    | **Botões de Chamada para Ação** | Adicione até 2 botões (opcional) |

    <Tip>
    Use marcadores como `{{1}}`, `{{2}}` na sua mensagem para personalizar o conteúdo com nomes de clientes, números de fatura ou outros dados dinâmicos.
    </Tip>
  </Step>

  <Step title="Criar Modelo">
    Revise seu modelo para verificar a precisão, depois clique no botão **Criar** para salvá-lo.

    ![Botão Criar Modelo](/images/1726058979.png)

    <Warning>
    Certifique-se de que seu modelo siga as políticas de mensagens do Meta para evitar rejeição durante o processo de aprovação.
    </Warning>
  </Step>
</Steps>

## Após a Criação

Uma vez criado, seu modelo será enviado ao Meta para aprovação. Após a aprovação:

<CardGroup cols={2}>
  <Card title="Acesso Rápido" icon="bolt">
    Acesse os modelos salvos instantaneamente através do menu de Resposta Rápida em qualquer conversa
  </Card>
  <Card title="Pronto para Transmissão" icon="paper-plane">
    Use modelos aprovados para campanhas de transmissão e mensagens em massa
  </Card>
</CardGroup>

## Casos de Uso

- **Suporte ao Cliente:** Crie modelos para respostas de suporte comuns
- **Alcance de Vendas:** Construa modelos de acompanhamento personalizados com marcadores
- **Atualizações de Pedidos:** Configure modelos de utilidade para notificações de envio e entrega
- **Lembretes de Compromissos:** Crie modelos de lembrete com marcadores de data/hora
