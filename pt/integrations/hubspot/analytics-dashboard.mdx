Este guia orienta você na criação de um painel de Analytics personalizado do HubSpot para rastrear suas métricas de mensagens do WhatsApp e dados de desempenho. Construa relatórios visuais que ajudam você a monitorar tempos de resposta, volumes de mensagens e eficácia da equipe.

## Criando Seu Painel de Analytics

<Steps>
  <Step title="Acessar Relatórios">
    Em sua conta do HubSpot, clique em **Reporting** (Relatórios) no menu de navegação principal.

    ![Clique em Relatórios](/images/hubspot-analytics-dashboard-1.png)
  </Step>
  <Step title="Abrir Relatórios">
    Clique em **Reports** (Relatórios) no menu suspenso.

    ![Clique em Relatórios](/images/hubspot-analytics-dashboard-2.png)
  </Step>
  <Step title="Criar Novo Relatório">
    Clique no botão **Create** (Criar) no canto superior direito.

    ![Clique em Criar](/images/hubspot-analytics-dashboard-3.png)
  </Step>
  <Step title="Selecionar Criar Relatório">
    Clique em **Create report** (Criar relatório) nas opções.

    ![Clique em Criar relatório](/images/hubspot-analytics-dashboard-4.png)
  </Step>
  <Step title="Continuar para o Construtor">
    Clique em **Next** (Próximo) para continuar para o construtor de relatórios.

    ![Clique em Próximo](/images/hubspot-analytics-dashboard-5.png)
  </Step>
  <Step title="Ativar Fontes de Dados">
    Na tela do Construtor de Relatórios Personalizados, ative **Add more data sources** (Adicionar mais fontes de dados) para acessar métricas adicionais.

    ![Ativar Adicionar mais fontes de dados](/images/hubspot-analytics-dashboard-6.png)
  </Step>
  <Step title="Continuar para Seleção de Dados">
    Clique em **Next** (Próximo) para prosseguir para a seleção de fontes de dados.

    ![Clique em Próximo](/images/hubspot-analytics-dashboard-7.png)
  </Step>
  <Step title="Pesquisar Fontes de Dados">
    Use a barra de pesquisa para encontrar as fontes de dados específicas que você deseja incluir em seu painel.

    ![Pesquisar fonte de dados](/images/hubspot-analytics-dashboard-8.png)
  </Step>
</Steps>

## Fontes de Dados do WhatsApp Disponíveis

Você pode escolher entre as seguintes fontes de dados de mensagens do WhatsApp:

| Fonte de Dados | Descrição |
|----------------|-----------|
| **Average Response Time** | Tempo médio para responder mensagens |
| **Client Replied** | Número de respostas de clientes recebidas |
| **Current Label** | Etiqueta atual atribuída à conversa |
| **Current Main Label** | Etiqueta de categoria principal |
| **Current Sub Label** | Etiqueta de subcategoria |
| **First Response Time** | Tempo até a primeira resposta |
| **Follow-ups** | Número de mensagens de acompanhamento enviadas |
| **Last AI Summary** | Resumo gerado por IA mais recente |
| **Last Message sent by** | Remetente da última mensagem |
| **Message Received** | Total de mensagens recebidas |
| **Message sent** | Total de mensagens enviadas |
| **Message you got** | Contagem de mensagens recebidas |
| **Message you sent** | Contagem de mensagens enviadas |
| **Time Since Last Client Message** | Tempo decorrido desde a última mensagem do cliente |
| **Total Messages** | Contagem combinada de mensagens |

<Note>
  A disponibilidade dessas fontes de dados depende da configuração da sua integração com o Eazybe e dos dados sincronizados do WhatsApp.
</Note>

## Construindo Seu Painel

<Steps>
  <Step title="Arrastar Dados para o Eixo">
    Clique na sua fonte de dados desejada e arraste-a para o eixo apropriado com base nos seus requisitos de relatório. Por exemplo, arraste **Total Messages** (Mensagens Totais) para visualizar o volume de mensagens.

    ![Arrastar fonte de dados para o eixo](/images/hubspot-analytics-dashboard-9.png)
  </Step>
  <Step title="Adicionar Mais Métricas">
    Continue arrastando e soltando fontes de dados adicionais com base nos requisitos do seu painel. Cada métrica pode ser configurada para diferentes tipos de visualização.

    ![Arrastar e soltar fontes de dados](/images/hubspot-analytics-dashboard-10.png)
  </Step>
  <Step title="Salvar Seu Relatório">
    Depois de configurar todas as métricas e visualizações desejadas, clique em **Save report** (Salvar relatório) para finalizar seu painel de analytics.

    ![Salvar o relatório](/images/hubspot-analytics-dashboard-11.png)
  </Step>
</Steps>

## Seu Painel de Analytics Está Pronto

Seu painel de Analytics personalizado do HubSpot agora está configurado e pronto para uso. Você pode acessar este painel a qualquer momento para monitorar o desempenho de suas mensagens do WhatsApp e rastrear as principais métricas que importam para sua equipe.

<Tip>
  **Dica Pro**: Agende revisões regulares do painel com sua equipe para identificar tendências, otimizar tempos de resposta e melhorar estratégias gerais de engajamento do cliente.
</Tip>
