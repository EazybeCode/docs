O EazyBe fornece várias APIs para ajudá-lo a automatizar e integrar mensagens do WhatsApp em seus fluxos de trabalho.

## APIs Atualmente Disponíveis

<CardGroup cols={2}>
  <Card title="API de Agendamento" icon="calendar" href="/pt/integrations/webhooks-custom/scheduler-api">
    Agende mensagens do WhatsApp programaticamente para entrega futura
  </Card>
  <Card title="API de Adicionar ID de Chat" icon="tag" href="/pt/integrations/webhooks-custom/add-chat-id">
    Adicione IDs de chat a etiquetas de negócio programaticamente
  </Card>
  <Card title="API de Criar Grupo" icon="users" href="/pt/integrations/webhooks-custom/create-group-and-add-chat-id">
    Crie grupos do WhatsApp e adicione IDs de chat via API
  </Card>
  <Card title="API de Usuários Ativos" icon="user-check" href="/pt/integrations/webhooks-custom/active-user-api">
    Obtenha uma lista de usuários recentemente ativos em sua organização
  </Card>
</CardGroup>
