## What Gets Synced

| Data Type | Direction | Description |
|-----------|-----------|-------------|
| Contacts | ↔️ Bidirectional | Create and link contacts |
| Conversations | → To CRM | Chat backup to timeline |
| Properties | ↔️ Bidirectional | Edit contact fields |
| Deals | ← From CRM | View deal info in WhatsApp |
| Tasks | → To CRM | Create follow-ups |
| Labels | ↔️ Bidirectional | Sync tags/labels |
