## Resumo

Este guia fornece instruções passo a passo sobre como vincular seus números API WABA a usuários específicos do HubSpot. O mapeamento bem-sucedido de usuários garante sincronização precisa de dados e gerenciamento eficiente do fluxo de trabalho entre suas plataformas.

<Note>
Antes de prosseguir, certifique-se de ter **permissões de administrador** tanto no Eazybe quanto no HubSpot para gerenciar mapeamentos de usuários.
</Note>

---

## Guia Passo a Passo

<Steps>
  <Step title="Navegue até a Página de Integrações">
    Vá para o painel do Eazybe e navegue até a seção **Integrações**.

    ![Navegue até a página de integrações](/images/waba-hubspot-1.png)

    <Tip>
    Você pode acessar a página de Integrações no menu principal na barra lateral esquerda.
    </Tip>
  </Step>

  <Step title="Acesse as Configurações de Integração do HubSpot">
    Encontre o cartão de integração do **HubSpot** e clique nele para acessar as configurações de integração.

    ![Acesse as configurações de integração do HubSpot](/images/waba-hubspot-2.png)
  </Step>

  <Step title="Selecione a Aba de Mapeamento de Usuários">
    Nas configurações de integração do HubSpot, localize e clique na aba **Mapeamento de Usuários**.

    ![Selecione a aba de mapeamento de usuários](/images/waba-hubspot-3.png)

    <Info>
    Esta aba permite que você atribua usuários específicos do HubSpot para gerenciar seus números API WABA.
    </Info>
  </Step>

  <Step title="Escolha um Usuário do HubSpot para Mapear">
    Selecione o **usuário do HubSpot** que deseja mapear para um número API WABA específico na lista de usuários disponíveis.

    ![Escolha um usuário do HubSpot para mapear](/images/waba-hubspot-4.png)
  </Step>

  <Step title="Selecione o Usuário Específico do HubSpot">
    Escolha o usuário específico do HubSpot no menu suspenso que deseja associar ao seu número API WABA.

    ![Selecione o usuário específico do HubSpot](/images/waba-hubspot-5.png)

    <Warning>
    Certifique-se de que o usuário selecionado tenha as permissões necessárias no HubSpot para gerenciar comunicações da API.
    </Warning>
  </Step>

  <Step title="Verifique o Mapeamento Bem-Sucedido do Usuário">
    Confirme o mapeamento e verifique se o usuário foi vinculado com sucesso ao número API WABA.

    ![Verifique o mapeamento bem-sucedido do usuário](/images/waba-hubspot-6.png)

    <Tip>
    Você pode modificar ou remover mapeamentos de usuários a qualquer momento na aba Mapeamento de Usuários.
    </Tip>
  </Step>
</Steps>

## Solução de Problemas

<Accordion>
  <AccordionItem title="Usuário não aparece na lista?">
    Certifique-se de que o usuário foi convidado para o seu portal do HubSpot e aceitou o convite. Apenas usuários ativos do HubSpot com as permissões apropriadas aparecerão na lista de mapeamento.
  </AccordionItem>
  <AccordionItem title="Mapeamento não está sendo salvo?">
    Verifique se você tem as permissões de administrador necessárias. Se o problema persistir, tente atualizar a página e tentar o mapeamento novamente.
  </AccordionItem>
  <AccordionItem title="Posso mapear múltiplos números API para um usuário?">
    Sim, um único usuário do HubSpot pode ser mapeado para múltiplos números API WABA. Isso é útil para membros da equipe que gerenciam múltiplos números de negócio.
  </AccordionItem>
</Accordion>
