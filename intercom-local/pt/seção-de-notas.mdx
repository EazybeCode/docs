Seção de Notas

Você tem algo importante acontecendo, pode esquecer onde terminou a última conversa.

Não se preocupe, com a seção EazyBe Notes você pode criar notas de acompanhamento e salvar mensagens importantes.

Mas você deve estar se perguntando, como?

Vamos descobrir.

- Vá para o contato para o qual deseja criar um acompanhamento.
- Clique no ícone HubSpot. O terceiro botão na barra superior o levará para a seção Notas.

  

3. Você pode escrever diretamente as notas de acompanhamento na caixa de texto abaixo e clicar em Salvar para salvar as notas.

  

4. Clique no botão e você poderá ver as notas que criou.
