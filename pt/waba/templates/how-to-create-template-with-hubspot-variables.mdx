## Visão Geral

Este guia mostra como criar modelos de mensagens que usam propriedades de contato do HubSpot como variáveis dinâmicas. Esses modelos permitem mensagens personalizadas do WhatsApp através da integração com a WhatsApp Business API, extraindo automaticamente dados do seu CRM HubSpot.

## Criando um Modelo com Variáveis do HubSpot

<Steps>
  <Step title="Navegar para Criar Modelo">
    Vá para a aba **Criar Modelo** e selecione um contato no menu suspenso.

    <Frame>
      <img src="/images/1911603921.png" alt="Navegar para a aba Criar Modelo" />
    </Frame>
  </Step>

  <Step title="Iniciar Criação do Modelo">
    Clique no botão **Criar Modelo** para começar.

    <Frame>
      <img src="/images/1911603922.png" alt="Clique no botão Criar Modelo" />
    </Frame>
  </Step>

  <Step title="Inserir Nome do Modelo">
    Digite um nome único para o modelo. Evite caracteres especiais no nome.

    <Frame>
      <img src="/images/1911603926.png" alt="Inserir nome do modelo" />
    </Frame>
  </Step>

  <Step title="Selecionar Tipo de Modelo">
    Escolha o tipo de modelo apropriado das opções disponíveis (Marketing, Utilidade, etc.).

    <Frame>
      <img src="/images/1911603929.png" alt="Selecionar tipo de modelo" />
    </Frame>
  </Step>

  <Step title="Escolher Idioma">
    Selecione seu idioma preferido para o modelo no menu suspenso.

    <Frame>
      <img src="/images/1911603936.png" alt="Escolher idioma do modelo" />
    </Frame>
  </Step>

  <Step title="Escrever Conteúdo da Mensagem">
    Preencha o conteúdo da sua mensagem no campo **Corpo da Mensagem**.

    <Frame>
      <img src="/images/1911603933.png" alt="Escrever conteúdo do corpo da mensagem" />
    </Frame>
  </Step>

  <Step title="Posicionar Cursor para Variável">
    Coloque seu cursor na posição exata onde você deseja inserir um valor de propriedade do HubSpot.

    <Frame>
      <img src="/images/1911603931.png" alt="Posicionar cursor para inserção de variável" />
    </Frame>
  </Step>

  <Step title="Selecionar Opção de Variável HubSpot">
    No menu suspenso disponível, selecione **Variável do HubSpot**.

    <Frame>
      <img src="/images/1911603927.png" alt="Selecionar opção Variável do HubSpot" />
    </Frame>
  </Step>

  <Step title="Explorar Propriedades do HubSpot">
    Clique em **Adicionar Variável** para abrir o navegador de propriedades do HubSpot.

    <Frame>
      <img src="/images/1911603934.png" alt="Clique em Adicionar Variável para explorar propriedades" />
    </Frame>
  </Step>

  <Step title="Buscar Propriedade">
    Use a função de busca para encontrar a propriedade do HubSpot desejada na lista.

    <Frame>
      <img src="/images/1911603925.png" alt="Buscar propriedade do HubSpot" />
    </Frame>
  </Step>

  <Step title="Inserir Variável">
    Clique na propriedade do HubSpot escolhida para inseri-la como variável na sua mensagem.

    <Frame>
      <img src="/images/1911603935.png" alt="Inserir propriedade do HubSpot como variável" />
    </Frame>
  </Step>

  <Step title="Adicionar Mais Variáveis (Opcional)">
    Repita os passos anteriores para adicionar variáveis adicionais do HubSpot conforme necessário.

    <Frame>
      <img src="/images/1911603928.png" alt="Adicionar variáveis adicionais" />
    </Frame>
  </Step>

  <Step title="Definir Valores de Fallback">
    Insira valores de fallback para cada variável. Esses valores padrão são usados quando os dados da propriedade do HubSpot não estão disponíveis para um contato.

    <Frame>
      <img src="/images/1911603932.png" alt="Definir valores de fallback para variáveis" />
    </Frame>
  </Step>

  <Step title="Enviar para Aprovação">
    Clique em **Enviar para Aprovação** para enviar seu modelo ao META para revisão.

    <Frame>
      <img src="/images/1911603930.png" alt="Enviar modelo para aprovação" />
    </Frame>
  </Step>
</Steps>

## Restrições Importantes

<Warning>
Tenha estas diretrizes em mente ao criar modelos com variáveis do HubSpot:

- **Posicionamento de variáveis**: Não coloque variáveis no início ou no final da sua mensagem
- **Equilibrar conteúdo**: Mantenha uma boa proporção entre quantidade de variáveis e conteúdo estático
- **Nomes únicos**: Cada nome de modelo deve ser único em sua conta
- **Comportamento de fallback**: Os valores de fallback são ativados quando os dados do proprietário do contato estão inacessíveis no HubSpot
</Warning>

<Note>
**Entendendo os Valores de Fallback**

O valor de fallback é o texto padrão que é exibido no lugar do valor da propriedade do HubSpot quando essa propriedade não está disponível ou está vazia para um contato específico. Sempre defina valores de fallback significativos para garantir que suas mensagens permaneçam coerentes.
</Note>
