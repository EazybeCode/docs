</Info>

## The Solution

WhatsApp messages automatically logged as Pipedrive activities.

<Frame caption="WhatsApp in Pipedrive activities">
  <img src="/images/pipedrive/chat-backup-solution.gif" alt="Chat backup" />
</Frame>

## Setup

<Steps>
  <Step title="Enable Chat Backup">
    **Settings** → **Pipedrive** → Enable **Backup chats**
  </Step>
  
  <Step title="Choose Activity Type">
    Messages logged as "WhatsApp" activity type
    
    <Frame>
      <img src="/images/pipedrive/activity-type.png" alt="Activity type" />
    </Frame>
  </Step>
</Steps>
