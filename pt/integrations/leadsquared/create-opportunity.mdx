Oportunidades representam negócios potenciais ou clientes que demonstram interesse no seu produto. Elas ajudam você a categorizar clientes com base no status das conversas, permitindo priorizar os mais importantes.

## Criação e Gerenciamento de Oportunidades

<Steps>
  <Step title="Abrir o Chat do WhatsApp do Lead">
    Acesse a conversa do WhatsApp do contato do LeadSquared para o qual você deseja criar uma oportunidade.
  </Step>

  <Step title="Acessar a Seção de Oportunidades">
    Navegue até a **visualização Mini-CRM** na barra lateral direita. A seção de Oportunidades aparece apenas se o lead já existir no LeadSquared.

    ![Seção de oportunidades no Mini-CRM](/images/1224408720.png)
  </Step>

  <Step title="Criar Oportunidade">
    Clique no botão **Criar Oportunidade** para começar a adicionar uma nova oportunidade para este lead.

    ![Botão Criar Oportunidade](/images/1224408957.png)
  </Step>

  <Step title="Editar Detalhes da Oportunidade">
    Preencha as informações necessárias da oportunidade, incluindo valor do negócio, estágio e outros detalhes relevantes.
  </Step>

  <Step title="Salvar Detalhes da Oportunidade">
    Clique no botão **Salvar** para finalizar e criar a oportunidade.

    ![Formulário Salvar Oportunidade](/images/1224409190.png)
  </Step>

  <Step title="Adicionar Mais Oportunidades">
    Repita o processo para criar oportunidades adicionais para o mesmo lead conforme necessário.
  </Step>
</Steps>

<Info>
  As oportunidades criadas no EazyBe são automaticamente sincronizadas com o LeadSquared e vinculadas ao lead específico.
</Info>
