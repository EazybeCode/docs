## Contact Sync

### How It Works

1. Open a WhatsApp chat
2. Eazybe searches Google Sheets by phone number
3. Matched contact data appears in sidebar
4. Create new contacts if not found

### Settings

| Setting | Description |
|---------|-------------|
| Auto-match | Link by phone number |
| Auto-create | Create new contacts |
| Update existing | Keep data synced |

<Warning>
Include country codes in phone numbers for best matching.
</Warning>
