## Extension Issues

<AccordionGroup>
  <Accordion title="Eazybe sidebar not appearing">
    1. Refresh WhatsApp Web page
    2. Click Eazybe icon in Chrome toolbar
    3. Check extension is enabled in chrome://extensions
    4. Disable conflicting extensions
    5. Clear browser cache
  </Accordion>
  
  <Accordion title="Extension keeps crashing">
    1. Update Chrome to latest version
    2. Reinstall Eazybe extension
    3. Disable other extensions temporarily
    4. Check available memory
  </Accordion>
</AccordionGroup>

## Sync Issues

<AccordionGroup>
  <Accordion title="Contacts not syncing to CRM">
    1. Verify CRM connection in Settings
    2. Check phone number format
    3. Ensure contact has phone field filled
    4. Wait 5 minutes for sync cycle
    5. Check last sync time in workspace
  </Accordion>
  
  <Accordion title="Chat backup not appearing in CRM">
    1. Confirm backup is enabled in settings
    2. Verify contact is linked to CRM
    3. Check sync frequency settings
    4. Look in correct timeline section
  </Accordion>
</AccordionGroup>

## Connection Issues

<AccordionGroup>
  <Accordion title="WhatsApp Web keeps disconnecting">
    1. Check phone internet connection
    2. Keep WhatsApp app open on phone
    3. Disable battery optimization for WhatsApp
    4. Avoid using WhatsApp Web on multiple devices
  </Accordion>
