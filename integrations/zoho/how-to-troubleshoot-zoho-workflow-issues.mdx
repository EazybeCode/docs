### To use workflows successfully with Zoho

- Ensure the **user has the Zoho workflow integration installed**.


- Once installed, you can proceed with creating and managing workflows.
​

  <img src="/images/1677035061.png" alt="" />
</Frame>


### Workflow messages in Zoho are always sent from:

- The **contact owner’s** WhatsApp number, or


- The **lead owner’s** WhatsApp number


<Frame>
  <img src="/images/1677310337.png" alt="" />
</Frame>


### For workflow messages to be delivered through Zoho, you must ensure:

- The **user has the Eazybe extension/app installed**.


- The **Zoho integration is properly connected** with Eazybe.


<Frame>
  <img src="/images/1677312091.png" alt="" />
</Frame>


### To avoid messages being sent to the wrong recipient:

- Make sure the **phone number includes the correct country code**.


- Double-check the number format before saving it in Zoho.


<Frame>
  <img src="/images/1677314079.png" alt="" />
</Frame>


### Media files have a size limit. To ensure successful delivery:

- Check that your **media file size does not exceed 15MB**.


- If it’s larger, compress or resize the file before attaching it to the workflow.
