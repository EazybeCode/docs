Creating WhatsApp templates directly from the Eazybe extension makes it easy to send approved messages or quick replies to your contacts without leaving WhatsApp Web.

## Overview

The Eazybe browser extension allows you to create message templates on-the-fly while chatting with customers. This is ideal for quickly setting up reusable messages for common responses.

<Info>
Templates created through the extension are submitted to Meta for approval and become available across all your conversations once approved.
</Info>

## How to Create Templates from the Extension

<Steps>
  <Step title="Access Quick Reply in Chat Input">
    Open the Eazybe extension on WhatsApp Web and locate the chat input section in any conversation. Click the **Quick Reply** option below the input field.

    ![Quick Reply Access](/images/1710847024.png)
  </Step>

  <Step title="Enable Template Creation Toggle">
    In the quick reply menu, turn **ON** the "Create Template" toggle. This activates the reusable message template creation mode.

    ![Create Template Toggle](/images/1710847407.png)
  </Step>

  <Step title="Fill Template Details">
    Complete the required fields for your template:

    ![Template Details Form](/images/1726023464.png)

    | Field | Description |
    |-------|-------------|
    | **Template Name** | A unique name for your template |
    | **Language** | Select the template language |
    | **Category** | Choose Marketing, Utility, or Authentication |
    | **Template Type** | Select the appropriate type |
    | **Message Content** | Write your template message |
    | **Footer** | Optional footer text |
    | **Call-to-Action Buttons** | Add up to 2 buttons (optional) |

    <Tip>
    Use placeholders like `{{1}}`, `{{2}}` in your message to personalize content with customer names, invoice numbers, or other dynamic data.
    </Tip>
  </Step>

  <Step title="Create Template">
    Review your template for accuracy, then click the **Create** button to save it.

    ![Create Template Button](/images/1726058979.png)

    <Warning>
    Ensure your template follows Meta's messaging policies to avoid rejection during the approval process.
    </Warning>
  </Step>
</Steps>

## After Creation

Once created, your template will be submitted to Meta for approval. After approval:

<CardGroup cols={2}>
  <Card title="Quick Access" icon="bolt">
    Access saved templates instantly via the Quick Reply menu in any conversation
  </Card>
  <Card title="Broadcast Ready" icon="paper-plane">
    Use approved templates for broadcast campaigns and bulk messaging
  </Card>
</CardGroup>

## Use Cases

- **Customer Support:** Create templates for common support responses
- **Sales Outreach:** Build personalized follow-up templates with placeholders
- **Order Updates:** Set up utility templates for shipping and delivery notifications
- **Appointment Reminders:** Create reminder templates with date/time placeholders
