## Genel Bakış

Pipedrive'ı Eazybe WhatsApp uzantınızla entegre etmek basit ve hızlıdır. Başlamadan önce, sorunsuz bir kurulum süreci için aşağıdaki ön koşulları karşıladığınızdan emin olun.

<Note>
  Tüm ön koşulları karşılamak sorunsuz bir entegrasyon deneyimi sağlar ve
  yaygın kurulum sorunlarını önler.
</Note>

---

## Gereksinimler Kontrol Listesi

<Steps>
  <Step title="Aktif Pipedrive Hesabı">
    Pipedrive hesabınızın doğru şekilde kurulduğundan ve çalıştığından emin olun. Entegrasyonu tamamlamak için Pipedrive portalınıza erişiminiz olması gerekir.

    <Info>
    Hem **ücretsiz** hem de **ücretli** Pipedrive planları Eazybe entegrasyonunu destekler.
    </Info>

  </Step>

  <Step title="Eazybe WhatsApp Uzantısını Yükleyin">
    Eazybe WhatsApp uzantınızın doğru şekilde yüklendiğinden ve WhatsApp Web'e bağlı olduğundan emin olun.

    Henüz uzantıyı yüklemediyseniz, indirmek için [Chrome Web Mağazası](https://chromewebstore.google.com/detail/whatsapp-ai-agents-with-c/clgficggccelgifppbcaepjdkklfcefd) adresini ziyaret edin.

  </Step>

  <Step title="Şirket Etki Alanı">
    Entegrasyon için Pipedrive Şirket Etki Alanınız gerekecek.

    <img src="/images/pipedrive-04-company-domain.png" alt="Pipedrive Şirket Etki Alanı" />

    Bu, Pipedrive URL'nizdeki alt alandır. Pipedrive'a giriş yaptığınızda adres çubuğuna bakın — şöyle okunacak:

    `https://sirketiniz.pipedrive.com`

    **.pipedrive.com**'dan önceki kısım Şirket Etki Alanınızdır.

  </Step>

  <Step title="Pipedrive API Anahtarı">
    Bağlantıyı yetkilendirmek için Pipedrive API Anahtarınız gerekecek.

    API Anahtarınızı almak için:

    <img src="/images/pipedrive-05-api-token.png" alt="Pipedrive API Anahtarı" />

    1. Pipedrive hesabınıza gidin
    2. **Kişisel Tercihler**'e tıklayın
    3. **API** sekmesine tıklayın
    4. Mevcut API Anahtarınızı kopyalayın veya yenisi oluşturun

  </Step>
</Steps>

---

## Hızlı Referans

| Gereksinim           | Ayrıntılar                                       |
| --------------------- | --------------------------------------------- |
| **Pipedrive Hesabı** | Aktif hesap (ücretsiz veya ücretli)                 |
| **Eazybe Uzantısı**  | Yüklü ve WhatsApp Web'e bağlı       |
| **Şirket Etki Alanı**    | Pipedrive alt alanınız                      |
| **API Anahtarı**         | Kişisel Tercihler → API sekmesinden oluşturuldu |
