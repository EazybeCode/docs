## Prerequisites

Before setting up WABA, you need:

<Check>Meta Business Account (Facebook Business Manager)</Check>
<Check>Phone number that can receive SMS/calls for verification</Check>
<Check>Eazybe subscription (Starter plan or higher)</Check>

## Setup Process

<Steps>
  <Step title="Access WABA Setup">
    In Eazybe workspace, go to **Settings** → **WhatsApp Business API** → **Add Number**.
  </Step>
  
  <Step title="Connect Meta Business">
    Follow the embedded signup flow:
    1. Log in to Facebook
    2. Create or select Meta Business Portfolio
    3. Accept WhatsApp Business Terms
  </Step>
  
  <Step title="Verify Phone Number">
    Enter the phone number you want to use.
    Choose SMS or voice call for OTP verification.
  </Step>
  
  <Step title="Configure Account">
    Set your business profile:
    - Business name
    - Category
    - Description
    - Website
  </Step>
  
  <Step title="Complete Setup">
    Your number is now connected to WABA.
    You can start creating templates and sending messages.
  </Step>
</Steps>

## Choosing Your Number

| Option | Description | Best For |
|--------|-------------|----------|
| New number | Fresh number for WABA only | Dedicated API line |
| Existing number | Migrate current WhatsApp | Keep existing contacts |
| Coexistence | Use same number for App + API | Best of both worlds |

<Info>
  We recommend Coexistence mode for most businesses. It lets you keep using the WhatsApp Business App while adding API capabilities.
</Info>

## After Setup

Once connected, you can:

1. **Create message templates** - Required for outbound messaging
2. **Fund your wallet** - Add credits for Meta conversation fees
3. **Set up broadcasts** - Send campaigns to your contacts
4. **Enable workflows** - Automate messages from your CRM
