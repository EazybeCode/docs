## Admin Management

### Granting Access

1. Go to Eazybe Workspace
2. Navigate to **Team** > **Members**
3. Invite and assign access

### Permission Levels

| Role | Access |
|------|--------|
| Admin | Full configuration |
| Member | Sync and view |
| Viewer | Read-only |
