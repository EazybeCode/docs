## Quais Permissões São Necessárias para a Integração?

Durante a configuração da integração com o HubSpot, as permissões necessárias serão solicitadas para garantir o funcionamento adequado da extensão Chrome do Eazybe para WhatsApp.

<Warning>
Seu **super-administrador** deve conceder duas permissões críticas para que a integração funcione corretamente:
- Permissão de **Arquivo**
- Permissão de **Apps Marketplace**
</Warning>

Essas permissões são essenciais para a sincronização e backup suaves dos seus chats do WhatsApp usando a extensão Chrome do Eazybe para WhatsApp.

## Completando a Integração

<Steps>
  <Step title="Revisar Permissões">
    Revise cuidadosamente todas as permissões listadas no diálogo de integração.
  </Step>

  <Step title="Reconhecer Permissões">
    Selecione as caixas de seleção para reconhecer cada permissão da lista fornecida.

    ![Botão Conectar App](/images/1175570125.png)
  </Step>

  <Step title="Conectar o App">
    Clique no botão **Conectar App** para completar a integração com sucesso.
  </Step>
</Steps>
