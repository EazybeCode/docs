## What is Broadcasting?

Broadcasting allows you to send the **same message to multiple contacts** simultaneously using approved templates. Unlike WhatsApp Business App's 256-contact limit, WABA broadcasts can reach **thousands of recipients**.

## Broadcast vs Regular Messages

| Feature | Regular Message | Broadcast |
|---------|-----------------|-----------|
| Recipients | 1 | Unlimited |
| Template Required | Only outside 24hr | Always |
| Personalization | Manual | Variables auto-filled |
| Tracking | Basic | Full analytics |
| Scheduling | No | Yes |

## Requirements

Before sending broadcasts:

- ✅ WABA account connected
- ✅ At least one approved message template
- ✅ Sufficient wallet balance
- ✅ Contact list with opted-in recipients
