Eazybe is a Chrome extension that enhances WhatsApp Web with productivity and CRM integration features for sales and support teams. Key capabilities include CRM mini-views, shared team inboxes, contact labels, and data synchronization to Google Drive or CRM systems.

## Data Handling & Storage

| Category | Details |
|----------|---------|
| **Message Storage** | Eazybe does not store messages. All chats remain within WhatsApp. |
| **CRM Data** | Displayed via APIs; credentials not stored; OAuth preferred |
| **Backups** | Stored in user's Google Drive |
| **Attachments** | Media stored in Google Drive when backup enabled |
| **Server Storage** | Data remains client-side or within user's cloud services |

## Encryption & Infrastructure

<AccordionGroup>
  <Accordion title="Data in Transit">

  </Accordion>
  <Accordion title="Stored Backups">

</AccordionGroup>

## Chat Backup Defaults

<Warning>
When Team Inbox is enabled, all WhatsApp chats are backed up by default to Google Drive.
</Warning>

Users can control backup settings via:
- **"Don't Sync" labels** for specific contacts
- **Admin controls** for organization-wide sync rules

## Cloud WhatsApp Connectivity

Optional feature enabling scheduled/automated messaging when browser is closed:

- No WhatsApp data stored on Eazybe servers
- Authentication via WhatsApp's QR code login
- Sessions can be disconnected anytime from extension

## IT Setup Requirements

<Steps>
  <Step title="Browser Requirements">
    Google Chrome browser required with WhatsApp Web login in the same Chrome profile.
  </Step>
  <Step title="Extension Permissions">
    Enable Chrome extension permissions for Eazybe.
  </Step>
  <Step title="Optional: CRM Integration">
    Configure CRM API keys or OAuth credentials if using CRM features.
  </Step>
  <Step title="Optional: Google Drive">
    Grant Google Drive access for workspace admins if using backup features.
  </Step>
  <Step title="Firewall Configuration">
    Allow HTTPS to `web.eazybe.com`, `api.eazybe.com`, and your CRM endpoints.
  </Step>
</Steps>

<Tip>
Eazybe does not require software installation on user machines - just a lightweight Chrome extension. We recommend using test accounts for safe evaluation.
</Tip>
