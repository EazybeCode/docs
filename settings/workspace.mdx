## Overview

Workspace settings control organization-wide configurations for your Eazybe account.

## Accessing Settings

1. Click **Settings** (gear icon) in sidebar
2. Select **Workspace**

## General Settings

### Organization Name

- Display name for your workspace
- Shown in team invites and reports

### Logo

- Upload your company logo
- Appears in Revenue Inbox header

### Time Zone

- Set default timezone for scheduling
- Affects analytics and reports

## WhatsApp Settings

| Setting | Description |
|---------|-------------|
| **Default Sender** | Which number sends automated messages |
| **Business Hours** | When team is available |
| **Away Message** | Auto-reply outside hours |

## Sync Settings

### Google Drive Backup

- **Enable/Disable**: Toggle chat backup
- **Folder**: Choose backup location
- **Frequency**: Real-time or batched

### CRM Sync

- **Auto-create contacts**: Create CRM records for new chats
- **Sync attachments**: Include media in CRM
- **Activity types**: What gets logged

## Notification Settings

| Notification | Options |
|--------------|---------|
| **Unreplied alerts** | Frequency, recipients |
| **Daily digest** | Time, recipients |
| **New lead alerts** | Conditions, channels |

## Advanced

### Data Retention

- **Chat history**: How long to keep
- **Analytics data**: Retention period

### API Access

- View API keys
- Generate new keys
- Set rate limits

## Changes

Changes save automatically. Some settings require page refresh to take effect.
