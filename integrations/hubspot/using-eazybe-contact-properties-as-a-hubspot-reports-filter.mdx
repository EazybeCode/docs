title: "Using EazyBe Contact properties as a HubSpot report's filter"
description: ""
---

You can sync different contact properties for your HubSpot, which are:


- Last message sent by the contact.


- Last message received by the contact.


- Last WhatsApp messages sync time.


You can use these properties to generate reports for your own use case.


To create the reports, the steps that you have to follow are:


- Ensure that your HubSpot is integrated with EazyBe WhatsApp extension.


- Click on the Create Report button and select Custom Report Builder.


- Select Contact as the data source and choose the necessary objects that you want to be included in your reports.


- Select Activities and click Next.


- Select the EazyBe Properties that you want to use and want a report to get created for.


- Select all the required fields.


- Click on Save Report.


- Enter the name for the report and click on Save and Add.
