Se você está vendo uma notificação sobre limite de API excedido, este artigo explica os limites de taxa em vigor ao trabalhar com a integração LeadSquared.

## Informações sobre o Limite de Taxa

<Warning>
  Existe um limite de taxa de **5 chamadas de API por segundo** ao trabalhar com a integração LeadSquared.
</Warning>

O sistema EazyBe foi otimizado para lidar com essa restrição de forma eficiente. No entanto, quando você recebe uma notificação sobre limites de API, isso indica que o limite de 5 chamadas por segundo foi excedido.

## Por Que Isso Acontece

A notificação de limite de taxa de API pode aparecer quando:

- Múltiplas conversas do WhatsApp estão sendo sincronizadas simultaneamente
- Ações rápidas estão sendo realizadas no painel Mini-CRM
- Vários membros da equipe estão usando ativamente a integração ao mesmo tempo

<Info>
  Esta é uma limitação imposta pela API do LeadSquared, não pelo EazyBe. O limite de taxa ajuda a garantir um serviço estável para todos os usuários.
</Info>

## O Que Fazer

<Tip>
  Se você encontrar frequentemente a notificação de limite de API:
  - Aguarde alguns segundos antes de tentar novamente sua ação
  - Espaçe suas atividades ao realizar múltiplas operações
  - O sistema tentará automaticamente novamente as solicitações falhadas quando o limite for redefinido
</Tip>
