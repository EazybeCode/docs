## Overview

Quick Replies are pre-saved message templates that let you respond to common queries instantly with a single click or shortcut.

## Creating Quick Replies

<Steps>
  <Step title="Open Quick Replies">
    Click the **Quick Replies** icon in the Eazybe sidebar
  </Step>
  <Step title="Click Create">
    Click **+ Create Quick Reply**
  </Step>
  <Step title="Enter Details">
    - **Title**: Searchable name (e.g., "Pricing Info")
    - **Shortcut**: Quick trigger (e.g., `/price`)
    - **Message**: The response text
    - **Attachments**: Optional files
  </Step>
  <Step title="Set Visibility">
    - **Personal**: Only you can see
    - **Shared**: Team can use
  </Step>
  <Step title="Save">
    Click **Save Quick Reply**
  </Step>
</Steps>

## Using Quick Replies

### Method 1: Click to Insert

1. Open Quick Replies panel
2. Search or browse
3. Click to insert into message field
4. Edit if needed, then send

### Method 2: Shortcut

1. Type `/` in message field
2. Type shortcut (e.g., `/price`)
3. Press Enter to insert
4. Send

### Method 3: Search

1. Type `/` followed by search term
2. Results filter in real-time
3. Select with arrow keys
4. Press Enter

## Best Practices

<Tip>
  Use descriptive shortcuts that are easy to remember: `/greeting`, `/pricing`, `/hours`
</Tip>

### Organizing Quick Replies

| Category | Example Shortcuts |
|----------|-------------------|
| Greetings | `/hi`, `/welcome`, `/thanks` |
| Sales | `/pricing`, `/demo`, `/proposal` |
| Support | `/hours`, `/refund`, `/shipping` |
| Follow-up | `/followup`, `/checkin`, `/reminder` |

## Variables in Quick Replies

Use placeholders for personalization:

| Variable | Replaced With |
|----------|---------------|
| `{{name}}` | Contact's name |
| `{{company}}` | Company name |
| `{{date}}` | Current date |

## Sharing Quick Replies

### Make Reply Shared

1. Edit quick reply
2. Change visibility to **Shared**
3. Save

### Team Access

- Shared replies visible to all team members
- Admins can manage all shared replies
- Members can only edit their own
