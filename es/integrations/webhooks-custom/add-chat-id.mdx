---
title: "Agregar ID de Chat"
description: "Agrega IDs de chat a etiquetas de negocio programáticamente usando la API de EazyBe"
---

La API de Agregar ID de Chat te permite agregar chats específicos a etiquetas de negocio. Verifica si la sesión en la nube está activa, recupera las etiquetas de negocio disponibles y asocia los chats especificados con una etiqueta determinada.

<Warning>
No uses esta API para spam o mensajes masivos. El uso indebido puede resultar en bloqueo de cuenta.
</Warning>

## Endpoint

```
POST https://api.eazybe.com/v2/cloud/business-label
```

## Autenticación

Necesitas obtener el **Token de Autenticación de la Organización** desde tu Workspace de EazyBe.

<Steps>
  <Step title="Accede al Workspace de EazyBe">
    Ve a [workspace.eazybe.com](https://workspace.eazybe.com)
  </Step>

  <Step title="Navega a la Página de Organización">
    Un administrador necesita acceder a la página de Organización.
  </Step>

  <Step title="Copia el Token de Autenticación">
    El Token de Autenticación de la Organización se mostrará en esta página. Cópialo para usar con la API.
  </Step>
</Steps>

## Headers

| Header | Valor | Descripción |
|--------|-------|-------------|
| `Content-Type` | `application/json; charset=UTF-8` | Especifica formato JSON |
| `Authorization` | `Bearer {token}` | Token de Autenticación de la Organización |

## Payload de la Solicitud

La API requiere los siguientes campos:

| Campo | Tipo | Descripción |
|-------|------|-------------|
| `labelName` | string | Nombre de la etiqueta de negocio para agregar chats |
| `contacts` | array | Lista de números de teléfono de contactos (formato internacional) |

### Ejemplo de Payload

```json
{
  "labelName": "Priority Clients",
  "contacts": [
    "911234567891",
    "911234567892",
    "911234567893"
  ]
}
```

## Ejemplo de Código (Node.js)

```javascript
const axios = require('axios');

const config = {
  method: 'post',
  url: 'https://api.eazybe.com/v2/cloud/business-label',
  headers: {
    'Authorization': 'Bearer YOUR_ORGANIZATION_AUTH_TOKEN',
    'Content-Type': 'application/json; charset=UTF-8'
  },
  data: {
    labelName: 'Priority Clients',
    contacts: [
      '911234567891',
      '911234567892',
      '911234567893'
    ]
  }
};

axios(config)
  .then(response => console.log(response.data))
  .catch(error => console.error(error));
```

## Respuesta

### Respuesta Exitosa (200 OK)

```json
{
  "status": true,
  "data": {
    "message": "Contacts added successfully."
  }
}
```
