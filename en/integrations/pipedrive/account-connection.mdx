## Overview

Connecting Eazybe to your Pipedrive account enables powerful WhatsApp integration, allowing you to manage conversations, sync contacts, and streamline your sales workflow directly from Pipedrive.

<Note>
Before proceeding, ensure you have completed all the [prerequisites](/en/integrations/pipedrive/prerequisites) for the Pipedrive integration.
</Note>

---

## Step-by-Step Guide

<Steps>
  <Step title="Open WhatsApp Web with Eazybe Extension">
    Open WhatsApp Web and ensure the Eazybe extension is installed and active.

    <!-- TODO: Add screenshot of WhatsApp Web with Eazybe extension -->
    <img src="/images/TODO-pipedrive-whatsapp-web.png" alt="WhatsApp Web with Eazybe Extension" />
  </Step>

  <Step title="Navigate to Integrations">
    Click on the **CRM/Integrations** button in the Eazybe panel.

    <!-- TODO: Add screenshot of CRM/Integrations button -->
    <img src="/images/TODO-pipedrive-integrations-button.png" alt="CRM/Integrations Button" />
  </Step>

  <Step title="Select Pipedrive">
    Navigate to **Pipedrive** from the available CRM options and click on **Connect**.

    <!-- TODO: Add screenshot of Pipedrive option in integrations list -->
    <img src="/images/TODO-pipedrive-select-integration.png" alt="Select Pipedrive Integration" />
  </Step>

  <Step title="Enter Your Company Domain">
    Enter your Pipedrive Company Domain.

    <!-- TODO: Add screenshot of Company Domain input field -->
    <img src="/images/TODO-pipedrive-enter-domain.png" alt="Enter Company Domain" />

    <Info>
    Your Company Domain is the subdomain in your Pipedrive URL (e.g., `yourcompany` from `https://yourcompany.pipedrive.com`)
    </Info>
  </Step>

  <Step title="Enter Your API Token">
    Paste your Pipedrive API Token and click on **Connect**.

    <!-- TODO: Add screenshot of API Token input field -->
    <img src="/images/TODO-pipedrive-enter-token.png" alt="Enter API Token" />

    <Tip>
    If you haven't generated your API Token yet, go to Pipedrive → Personal Preferences → API tab → Generate new token
    </Tip>
  </Step>

  <Step title="Verify Connection">
    Once connected, you'll see the Pipedrive integration active in your Eazybe panel.

    <!-- TODO: Add screenshot showing successful connection -->
    <img src="/images/TODO-pipedrive-connection-success.png" alt="Connection Successful" />

    <Check>
    Your Pipedrive account is now successfully connected to Eazybe!
    </Check>
  </Step>
</Steps>
