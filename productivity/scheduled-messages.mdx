## How to Schedule a Message

<Steps>
  <Step title="Click Scheduler">
    Click the **Scheduler** button (clock icon) in the Eazybe toolbar while in any chat
  </Step>
  
  <Step title="Set Date & Time">
    Select when you want the message to be sent using the date/time picker
  </Step>
  
  <Step title="Set Recurrence (Optional)">
    Choose to repeat the message automatically:
    - **Daily** — Send every day at the same time
    - **Weekly** — Send on specific days each week
    - **Monthly** — Send on the same date each month
    - **Yearly** — Send on the same date each year
  </Step>
  
  <Step title="Compose Message">
    Type your message or select from your Quick Replies
    
    You can also:
    - Use dynamic fields from the bottom panel
    - Add the message to Quick Replies for future use
    - Attach files (max 20MB)
  </Step>
  
  <Step title="Schedule">
    Click **Schedule** to confirm. The message will be sent automatically at the specified time.
  </Step>
</Steps>

## Managing Scheduled Messages

### View All Scheduled Messages
Navigate to the Scheduler panel to see all pending messages with their scheduled times.

### Edit a Scheduled Message
1. Find the message in your scheduled list
2. Click **Edit** to modify content or timing
3. Save your changes

### Cancel a Scheduled Message
1. Find the message in your scheduled list
2. Click **Cancel** or **Delete**
3. Confirm cancellation

### Send Immediately
Changed your mind? Click the **Send Now** button to send a scheduled message right away instead of waiting.

## Best Timing Practices

<Tip>
**Optimal Sending Times:**
- **B2B Messages:** Tuesday-Thursday, 10am-12pm or 2pm-4pm
- **B2C Messages:** Evenings 6pm-9pm or weekends
- **Follow-ups:** 2-3 days after initial contact
- **Reminders:** 24 hours and 1 hour before appointments

Always consider your recipient's timezone!
</Tip>

## Scheduler API

For developers, Eazybe offers a Scheduler API to programmatically schedule messages from your own systems:

```javascript
POST https://api.eazybe.com/v2/scheduler/public-scheduler

Headers:
  Content-Type: application/json
  Authorization: {Organization Auth Token}

Body:
{
  "from": "your-email@company.com",
  "to": "919876543210",
  "message": "Hello! Following up on our conversation.",
  "name": "John Doe",
  "time": "2026-01-15T10:30:00.000Z"
}
```

### Getting Your Auth Token
1. Login to [Eazybe Workspace](https://workspace.eazybe.com)
2. Go to **Organization** settings
3. Copy your **Organization Auth Token**

<Warning>
**Use Responsibly:** Avoid using the API for spam or bulk messaging. Excessive automated messaging could result in your WhatsApp number being banned by Meta.
</Warning>

## FAQ

<AccordionGroup>
  <Accordion title="Do I need to keep my computer on for scheduled messages?">
    For Extension-based scheduling, yes—WhatsApp Web must be connected. For 24/7 scheduling without keeping your laptop open, enable Eazybe Cloud or use WABA integration.
  </Accordion>
  
  <Accordion title="What's the attachment limit for scheduled messages?">
    Attachments cannot exceed 20MB, and only one file can be attached per scheduled message.
  </Accordion>
  
  <Accordion title="Can I schedule messages to groups?">
    Yes, you can schedule messages to both individual contacts and group chats.
  </Accordion>
  
  <Accordion title="What happens if my computer is off when the message is scheduled to send?">
    The message won't be sent. It will remain in your scheduled queue and you'll need to reschedule it. Use Eazybe Cloud or WABA for reliable delivery.
  </Accordion>
  
  <Accordion title="Can I schedule the same message to multiple contacts?">
    Use the Bulk Messaging feature for sending to multiple contacts, or create recurring schedules for individual contacts.
  </Accordion>
</AccordionGroup>
