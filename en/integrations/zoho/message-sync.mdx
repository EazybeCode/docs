## Overview

Eazybe automatically syncs your WhatsApp conversations with Zoho CRM, ensuring all your customer communications are backed up and accessible within your CRM records.

## Automatic Sync Process

<Info>
  WhatsApp conversations with Zoho contacts or leads synchronize automatically **every 3 minutes**.
</Info>

The last synced date and time are displayed in the Eazybe extension. You can verify sync completion by clicking the icon to access the contact's Zoho page.

![Sync Status Display](/images/1223655076.png)

## Initial Integration Backup

<Note>
  Upon first Zoho integration, the system automatically provides chat backup for the **previous 3 days** of conversations.
</Note>

This ensures you don't lose any recent conversation history when setting up the integration.

## Where Messages Are Logged

### Notes Section

WhatsApp chats are backed up in the **Notes Section** of Contacts and Leads pages in Zoho CRM.

![Notes Section Backup](/images/1192590765.png)

### Deal Pages

Backup also appears across the **5 latest deals** associated with a contact on the Deal's page.

![Deal Page Backup](/images/1192593143.png)

## Attachments Backup

Attachments from WhatsApp are stored in both the **Notes** and **Attachments** sections of Zoho Leads and Contacts pages.

### In Notes Section

Links provide easy file access within the conversation context.

![Attachments in Notes](/images/1192596920.png)

### In Attachments Section

Files display in a tabular format with the following details:

| Column | Description |
|--------|-------------|
| **File Name** | Name of the attached file |
| **Attached By** | User who sent the attachment |
| **Date Added** | When the attachment was synced |
| **Size** | File size |

![Attachments Tab View](/images/1192693196.png)

![Attachments Table Details](/images/1192693197.png)
