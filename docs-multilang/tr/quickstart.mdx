## Quick Start Guide

<Steps>
  <Step title="Install Extension">
    Go to Chrome Web Store, search "Eazybe", click **Add to Chrome**
  </Step>
  <Step title="Open WhatsApp Web">
    Go to web.whatsapp.com and scan QR code
  </Step>
  <Step title="Connect CRM">
    Click **Integrations** → Select your CRM → Authorize
  </Step>
  <Step title="Start Using">
    Open any chat to see CRM data in the sidebar
  </Step>
</Steps>
