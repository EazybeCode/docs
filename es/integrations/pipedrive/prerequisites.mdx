## Resumen

Integrar Pipedrive con tu extensión Eazybe de WhatsApp es sencillo y rápido. Antes de comenzar, asegúrate de cumplir con los siguientes requisitos previos para un proceso de configuración fluido.

<Note>
  Cumplir con todos los requisitos previos garantiza una experiencia de integración fluida y
  evita problemas comunes de configuración.
</Note>

---

## Lista de Requisitos

<Steps>
  <Step title="Cuenta Pipedrive Activa">
    Asegúrate de que tu cuenta de Pipedrive esté configurada y funcionando correctamente. Necesitarás acceso a tu portal de Pipedrive para completar la integración.

    <Info>
    Tanto los planes **gratuitos** como **de pago** de Pipedrive admiten la integración con Eazybe.
    </Info>

  </Step>

  <Step title="Instalar la Extensión Eazybe de WhatsApp">
    Asegúrate de que tu extensión Eazybe de WhatsApp esté instalada correctamente y conectada a WhatsApp Web.

    Si aún no has instalado la extensión, visita la [Chrome Web Store](https://chromewebstore.google.com/detail/whatsapp-ai-agents-with-c/clgficggccelgifppbcaepjdkklfcefd) para descargarla.

  </Step>

  <Step title="Dominio de la Empresa">
    Necesitarás el Dominio de Empresa de Pipedrive para la integración.

    <img src="/images/pipedrive-04-company-domain.png" alt="Dominio de Empresa de Pipedrive" />

    Este es el subdominio en tu URL de Pipedrive. Cuando hayas iniciado sesión en Pipedrive, mira la barra de direcciones — leerá algo como:

    `https://tuempresa.pipedrive.com`

    La parte antes de **.pipedrive.com** es tu Dominio de Empresa.

  </Step>

  <Step title="Token de API de Pipedrive">
    Necesitarás tu Token de API de Pipedrive para autorizar la conexión.

    Para obtener tu Token de API:

    <img src="/images/pipedrive-05-api-token.png" alt="Token de API de Pipedrive" />

    1. Ve a tu cuenta de Pipedrive
    2. Haz clic en **Preferencias Personales**
    3. Haz clic en la pestaña **API**
    4. Copia tu Token de API existente o genera uno nuevo

  </Step>
</Steps>

---

## Referencia Rápida

| Requisito           | Detalles                                       |
| --------------------- | --------------------------------------------- |
| **Cuenta Pipedrive** | Cuenta activa (gratuita o de pago)                 |
| **Extensión Eazybe**  | Instalada y conectada a WhatsApp Web       |
| **Dominio de Empresa**    | Tu subdominio de Pipedrive                      |
| **Token de API**         | Generado desde Preferencias Personales → pestaña API |
