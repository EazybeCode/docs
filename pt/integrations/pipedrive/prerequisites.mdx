## Visão Geral

Integrar o Pipedrive com sua extensão Eazybe do WhatsApp é simples e rápido. Antes de começar, certifique-se de atender aos seguintes pré-requisitos para um processo de configuração tranquilo.

<Note>
  Atender a todos os pré-requisitos garante uma experiência de integração tranquila e
  evita problemas comuns de configuração.
</Note>

---

## Lista de Requisitos

<Steps>
  <Step title="Conta Pipedrive Ativa">
    Certifique-se de que sua conta Pipedrive esteja configurada e funcionando corretamente. Você precisará de acesso ao seu portal Pipedrive para concluir a integração.

    <Info>
    Tanto os planos **gratuitos** quanto **pagos** do Pipedrive suportam a integração com o Eazybe.
    </Info>

  </Step>

  <Step title="Instalar a Extensão Eazybe do WhatsApp">
    Certifique-se de que sua extensão Eazybe do WhatsApp esteja instalada corretamente e conectada ao WhatsApp Web.

    Se você ainda não instalou a extensão, visite a [Chrome Web Store](https://chromewebstore.google.com/detail/whatsapp-ai-agents-with-c/clgficggccelgifppbcaepjdkklfcefd) para baixá-la.

  </Step>

  <Step title="Domínio da Empresa">
    Você precisará do Domínio da Empresa do Pipedrive para a integração.

    <img src="/images/pipedrive-04-company-domain.png" alt="Domínio da Empresa do Pipedrive" />

    Este é o subdomínio na sua URL do Pipedrive. Quando você estiver logado no Pipedrive, olhe a barra de endereços — ela mostrará algo como:

    `https://suaempresa.pipedrive.com`

    A parte antes de **.pipedrive.com** é seu Domínio da Empresa.

  </Step>

  <Step title="Token de API do Pipedrive">
    Você precisará do seu Token de API do Pipedrive para autorizar a conexão.

    Para obter seu Token de API:

    <img src="/images/pipedrive-05-api-token.png" alt="Token de API do Pipedrive" />

    1. Vá para sua conta Pipedrive
    2. Clique em **Preferências Pessoais**
    3. Clique na aba **API**
    4. Copie seu Token de API existente ou gere um novo

  </Step>
</Steps>

---

## Referência Rápida

| Requisito           | Detalhes                                       |
| --------------------- | --------------------------------------------- |
| **Conta Pipedrive** | Conta ativa (gratuita ou paga)                 |
| **Extensão Eazybe**  | Instalada e conectada ao WhatsApp Web       |
| **Domínio da Empresa**    | Seu subdomínio do Pipedrive                      |
| **Token de API**         | Gerado em Preferências Pessoais → aba API |
