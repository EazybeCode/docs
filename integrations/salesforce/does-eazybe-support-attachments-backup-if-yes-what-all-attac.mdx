title: "Does EazyBe support attachments backup? If yes, what all attachments type are backed up in Salesforce?"
description: ""
---

Yes, EazyBe supports attachments backup. When syncing WhatsApp conversations with Salesforce, the following types of attachments are backed up:


- **Images**: Photos and screenshots shared in chats.


- **Documents**: Files such as PDFs, Word documents, and spreadsheets.


- **Audio Files**: Voice messages and other audio recordings.


- **Videos**: Video clips shared in conversations.


- **Other Files**: Any other types of files that may be shared in WhatsApp chats.


Attachments from WhatsApp conversations are stored in the **WhatsApp Chats by Eazybe** custom activity object of the Leads as well as Contacts page as given below:


Here, you can see a link created for an image. Clicking this link will download the attachment that has been backed up to Salesforce.
