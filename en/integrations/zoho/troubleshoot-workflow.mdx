This guide helps you diagnose and resolve common issues with Zoho workflows for WhatsApp messaging through EazyBe.

## Workflow Integration Requirements

To use workflows successfully with Zoho, ensure the following prerequisites are met:

<Steps>
  <Step title="Install Zoho Workflow Integration">
    Ensure the user has the Zoho workflow integration installed and properly configured.

    ![Workflow Integration](/images/1677035061.png)
  </Step>

  <Step title="Create and Manage Workflows">
    Once installed, proceed with creating and managing workflows through the Zoho automation settings.
  </Step>
</Steps>

## Message Sender Configuration

<Info>
  Workflow messages in Zoho are always sent from either the **contact owner's WhatsApp number** or the **lead owner's WhatsApp number**. Make sure the appropriate owner is assigned to each record.
</Info>

![Message Sender Configuration](/images/1677310337.png)

## Delivery Requirements

For workflow messages to be delivered successfully through Zoho, ensure the following:

<CardGroup cols={2}>
  <Card title="EazyBe Extension" icon="puzzle-piece">
    The user must have the EazyBe extension/app installed on their device
  </Card>
  <Card title="Zoho Integration" icon="link">

## Avoiding Wrong Recipients

To prevent messages being sent to the wrong recipient:

<Steps>
  <Step title="Include Country Code">
    Make sure the phone number includes the correct country code (e.g., +1 for USA, +91 for India).
  </Step>

  <Step title="Verify Number Format">
    Double-check the number format before saving it in Zoho. Remove any spaces, dashes, or special characters.
  </Step>
</Steps>

<Warning>
  Incorrect phone number formatting is one of the most common causes of workflow message delivery failures.
</Warning>

## Media File Size Limits

Media files have a size limit. To ensure successful delivery:

<Steps>
  <Step title="Check File Size">
    Verify that your media file size does not exceed **15MB**.
  </Step>

  <Step title="Compress Large Files">
    Compress or resize larger files before attaching them to workflows.
  </Step>
</Steps>

![Media File Requirements](/images/1677314079.png)

<Tip>
  Use online compression tools or reduce image resolution to keep file sizes under the 15MB limit while maintaining acceptable quality.
</Tip>
