## How to Schedule a Message

<Steps>
  <Step title="Click on the Scheduler Button">
    Click on the **Scheduler** button to open the scheduling interface.

    ![Scheduler Button](/images/1112605613.png)
  </Step>
  <Step title="Select Date and Time">
    Choose the date and time when you want your message to be sent.

    ![Select Date Time](/images/1112605768.png)
  </Step>
  <Step title="Configure Message Repetition">
    You can choose to repeat the message after a specified time interval for recurring communications.

    ![Configure Repetition](/images/1112605870.png)
  </Step>
  <Step title="Add to Quick Reply (Optional)">
    Select the fields from the bottom and check the **Add to Quick Reply** checkbox if you want to save this message for future use.

    ![Add to Quick Reply](/images/1112605985.png)

    ![Quick Reply Options](/images/1112606156.png)
  </Step>
  <Step title="Schedule and Send">
    The message will be scheduled and will be sent to the contact at the specified time.

    ![Schedule Message](/images/1112607191.png)
  </Step>
</Steps>

## Frequently Asked Questions

<AccordionGroup>
  <Accordion title="Is adding the message to Quick Reply mandatory?">
    No, adding the message to Quick Replies is optional. It is provided for ease of access to frequently sent messages.
  </Accordion>
  <Accordion title="What are the file attachment limits?">
    The file size for attachments in scheduled messages cannot exceed **20MB**, and only one file can be attached per message.
  </Accordion>
  <Accordion title="Can I edit a scheduled message before it's sent?">
    Yes, you can edit the scheduled message before sending it. Additionally, you have the option to send it immediately if needed.
  </Accordion>
  <Accordion title="Can I view or edit scheduled messages after setting them?">
    Yes, you can view and edit scheduled messages. The interface allows you to manage and make changes to your scheduled messages at any time.
  </Accordion>
  <Accordion title="How do I set up recurring messages?">
    In Step 4, you have the option to set the message as a Recurring Message. Choose the frequency - Days, Weeks, Months, or Years - based on your needs.
  </Accordion>
</AccordionGroup>

<Tip>
Use scheduled messages for follow-ups, reminders, or greetings to ensure timely communication with your contacts.
</Tip>
