Bu kılavuzda, WhatsApp müşteri destek iş akışınızı optimize etmek için Freshdesk'i EazyBe uzantısı ile nasıl entegre edeceğinizi öğreneceksiniz.

## Ön Koşullar

Başlamadan önce, şunlara sahip olduğunuzdan emin olun:

<CardGroup cols={2}>
  <Card title="En Son EazyBe Uzantısı" icon="puzzle-piece">
    EazyBe WhatsApp uzantısının en son sürümünün yüklü olduğundan emin olun.
  </Card>
  <Card title="Freshdesk Hesabı" icon="headset">
    API anahtarları oluşturmak için yönetici erişimine sahip aktif bir Freshdesk hesabı.
  </Card>
</CardGroup>

## Freshdesk Kimlik Bilgilerinizi Alma

EazyBe'yi bağlamadan önce, Freshdesk alan adınızı ve API anahtarınızı almanız gerekir.

### Adım 1: Alan Adı/Helpdesk Adınızı Çıkarın

Alan Adı/Helpdesk adınız aşağıda gösterildiği gibi Freshdesk URL'nizden elde edilebilir:

![Freshdesk Alan Adı Çıkarma](/images/1318277663.png)

### Adım 2: Hesap Ayarlarına Gidin

Freshdesk panonuzun sağ üst köşesinde bulunan **Hesabım** simgesine tıklayın.

![Hesabım Ayarları](/images/1318278504.png)

### Adım 3: API Anahtarınızı Oluşturun

Profil ayarları sayfanızı açın ve API anahtarınızı almak için CAPTCHA doğrulamasını tamamlayın.

![API Anahtarı Oluşturma](/images/1318278989.png)

<Note>
  API anahtarınızı güvende tutun ve herkese açık olarak paylaşmayın. Entegrasyonu tamamlamak için bu anahtara ihtiyacınız olacak.
</Note>

## Freshdesk'i EazyBe'ye Bağlama

<Steps>
  <Step title="WhatsApp Web'i Açın">
    WhatsApp Web'e erişin ve EazyBe uzantısının aktif olduğundan emin olun.
  </Step>
  <Step title="Entegrasyonlara Erişin">
    WhatsApp Web'deki sağ kenar çubuğundaki **Entegrasyonlar** düğmesine tıklayın.
  </Step>
  <Step title="Entegrasyon Seçeneklerini Görüntüleyin">
    Entegrasyon açılır penceresini görüntülemek için düğmeye tıklayın.
  </Step>
  <Step title="Freshdesk'i Seçin">
    **Freshdesk** bölümünü bulun ve **Bağlan**'a tıklayın.
  </Step>
  <Step title="Hesabınızı Seçin">
    Seçeneklerden Freshdesk hesabınızı seçin.
  </Step>
  <Step title="Hesap Bilgilerini Girin">
    Daha önce elde ettiğiniz Freshdesk alan adınızı ve API anahtarınızı girin.
  </Step>
  <Step title="Entegrasyonu Onaylayın">
    Tamamlandıktan sonra, entegrasyonun başarılı olduğuna dair onay alacaksınız.
  </Step>
</Steps>

<Tip>
  Başarılı entegrasyondan sonra, Freshdesk biletlerini doğrudan WhatsApp konuşmalarınızdan yönetebilirsiniz.
</Tip>
