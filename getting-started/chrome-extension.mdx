## System Requirements

Before installing Eazybe, ensure your system meets these requirements:

| Requirement | Minimum | Recommended |
|-------------|---------|-------------|
| Browser | Chrome 88+ | Chrome 120+ |
| Operating System | Windows 10, macOS 10.14, Linux | Latest versions |
| RAM | 4 GB | 8 GB+ |
| Internet | Stable connection | Broadband |

<Warning>
  Eazybe only works with Google Chrome. Other browsers (Firefox, Safari, Edge) are not currently supported.
</Warning>

## Installation Steps

<Steps>
  <Step title="Open Chrome Web Store">
    Navigate to the [Eazybe Chrome Extension](https://chromewebstore.google.com/detail/eazybe) in the Chrome Web Store.
    
    Alternatively, search for "Eazybe" in the Chrome Web Store.
  </Step>
  
  <Step title="Add to Chrome">
    Click the blue **Add to Chrome** button.
    
    <Frame>
      <img src="/images/chrome-add-extension.png" alt="Add to Chrome button" />
    </Frame>
  </Step>
  
  <Step title="Confirm Installation">
    A popup will appear asking for permissions. Click **Add extension** to confirm.
    
    Eazybe requires these permissions:
    - Read and change data on web.whatsapp.com
    - Display notifications
    - Storage for your preferences
  </Step>
  
  <Step title="Pin the Extension">
    Click the **puzzle icon** in Chrome's toolbar (top-right).
    
    Find Eazybe in the list and click the **pin icon** to keep it visible.
    
    <Frame>
      <img src="/images/chrome-pin-extension.png" alt="Pin extension" />
    </Frame>
  </Step>
  
  <Step title="Verify Installation">
    You should see the Eazybe icon in your Chrome toolbar. 
    
    A green checkmark indicates the extension is active.
  </Step>
</Steps>

## Activating Eazybe

After installation, activate Eazybe on WhatsApp Web:

<Steps>
  <Step title="Open WhatsApp Web">
    Go to [web.whatsapp.com](https://web.whatsapp.com)
  </Step>
  
  <Step title="Log in to WhatsApp">
    Scan the QR code with your phone's WhatsApp app.
  </Step>
  
  <Step title="Wait for Sidebar">
    The Eazybe sidebar will automatically appear on the right side of WhatsApp Web within 2-3 seconds.
  </Step>
</Steps>

<Tip>
  If the sidebar doesn't appear automatically, click the Eazybe icon in your Chrome toolbar to manually activate it.
</Tip>

## Troubleshooting Installation

<AccordionGroup>
  <Accordion title="Extension not appearing in toolbar">
    1. Click the puzzle icon in Chrome's toolbar
    2. Look for Eazybe in the list
    3. If not found, try reinstalling from Chrome Web Store
    4. Check if Chrome extensions are enabled in `chrome://extensions`
  </Accordion>
  
  <Accordion title="Sidebar not showing on WhatsApp Web">
    1. Refresh the WhatsApp Web page
    2. Click the Eazybe extension icon
    3. Clear browser cache and try again
    4. Disable other WhatsApp-related extensions that might conflict
  </Accordion>
  
  <Accordion title="Permission errors">
    1. Go to `chrome://extensions`
    2. Find Eazybe and click **Details**
    3. Ensure **Site access** is set to "On all sites" or at least includes web.whatsapp.com
    4. Toggle the extension off and on again
  </Accordion>
  
  <Accordion title="Extension crashing or not loading">
    1. Update Chrome to the latest version
    2. Remove and reinstall the Eazybe extension
    3. Disable other extensions temporarily to check for conflicts
    4. Contact support if the issue persists
  </Accordion>
</AccordionGroup>

## Updating the Extension

Chrome automatically updates extensions, but you can manually check for updates:

1. Go to `chrome://extensions`
2. Enable **Developer mode** (top-right toggle)
3. Click **Update** button

<Info>
  We recommend keeping Eazybe updated to access the latest features and security improvements.
</Info>

## Uninstalling

If you need to remove Eazybe:

1. Right-click the Eazybe icon in your toolbar
2. Select **Remove from Chrome**
3. Confirm removal

<Warning>
  Uninstalling the extension will stop chat sync. Your previously synced data remains in your CRM.
</Warning>
