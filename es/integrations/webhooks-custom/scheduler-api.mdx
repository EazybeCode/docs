La API de Programacion permite a los usuarios organizacionales autenticados programar mensajes de WhatsApp de forma programatica. La API valida la membresia del usuario en la organizacion, verifica los detalles requeridos del mensaje y maneja problemas de permisos.

<Warning>
No uses esta API para spam o mensajes masivos. El uso indebido puede resultar en bloqueo de cuenta.
</Warning>

## Endpoint

```
POST https://api.eazybe.com/v2/scheduler/public-scheduler
```

Necesitas obtener el **Token de Autenticacion de la Organizacion** desde tu Workspace de EazyBe.

## Payload de la Solicitud

La API requiere los siguientes campos:

| Campo | Tipo | Descripcion |
|-------|------|-------------|
| `from` | string | Direccion de correo organizacional del remitente |
| `to` | string | Numero de telefono del receptor (formato internacional) |
| `message` | string | Contenido de texto a enviar |
| `name` | string | Nombre del receptor para notificaciones |
| `time` | string | Marca de tiempo de entrega programada (formato ISO 8601) |

### Ejemplo de Payload

```json
{
  "from": "hello@eazybe.com",
  "to": "919876543210",
  "message": "Hello World",
  "name": "Vats",
  "time": "2024-03-01T17:25:00.000Z"
}
```

## Ejemplo de Codigo (Node.js)

```javascript
const axios = require('axios');

const config = {
  method: 'post',
  url: 'https://api.eazybe.com/v2/scheduler/public-scheduler',
  headers: {
    'Authorization': 'Bearer YOUR_ORGANIZATION_AUTH_TOKEN',
    'Content-Type': 'application/json'
  },
  data: {
    from: 'hello@eazybe.com',
    to: '919876543210',
    message: 'Hello World',
    name: 'Vats',
    time: '2024-03-01T17:25:00.000Z'
  }
};

axios(config)
  .then(response => console.log(response.data))
  .catch(error => console.error(error));
```

## Estado de Respuesta

| Estado | Descripcion |
|--------|-------------|
| **Exito** | Mensaje programado exitosamente |
| **Fallo** | Email invalido, campos faltantes, o problemas de permisos |
