## Overview

The Flow Builder is a drag-and-drop interface for creating chatbot conversation flows without coding.

## Interface Components

### Canvas
The main workspace where you build flows:
- Drag nodes from the sidebar
- Connect nodes with arrows
- Zoom in/out to navigate

## Node Details

### Message Node
- **Text**: Message content (supports variables)
- **Media**: Attach image, document, or video
- **Delay**: Optional pause before sending

### Question Node
- **Question text**: What to ask
- **Variable**: Store response in variable
- **Validation**: Ensure response format

### Button Node
- **Button text**: Label for each option
- **Max buttons**: Up to 3 buttons
- **Each button**: Connects to different path

### Condition Node
- **If**: Variable equals/contains value
- **Then**: Follow this path
- **Else**: Follow alternative path

## Variables

Store and use data throughout the flow:

| Variable | Example | Usage |
|----------|---------|-------|
| `{{name}}` | John | Customer's name |
| `{{phone}}` | +1234567890 | Phone number |
| `{{last_message}}` | "Help" | Their last message |
| `{{custom_field}}` | Any value | Your custom data |

Before activating:

2. Simulate conversation as customer
3. Test all possible paths
4. Verify handoff works
5. Check message timing
