Once a label/funnel is created, you will see plenty of options.

- Click on the label/funnel name and it will open up the contacts in the same.
- Click on the dropdown as given below.
- You will see a list of options as given below.
- Add contacts to label/funnel - You can add the contacts.
- Mark All Read - It marks all the chats in the label/funnel as read.
- Edit label/funnel - You can edit the properties using this feature.
- Upload via excel - To upload the contacts using an excel spreadsheet.
- Download contacts - The file will be downloaded in a .xls format and will contain the contacts of the label/funnel.
- Broadcast messages - You can create a broadcast message to send to all the contacts in the label. Do you want to know how to create a broadcast message, click on this link to know. 
- In the right-hand side of the mid-bar you can also see a dropdown where you can switch the sub-labels/stages of the created label/funnel respectively.
