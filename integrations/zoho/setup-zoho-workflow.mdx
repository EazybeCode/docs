1. The client should have the Zoho workflow integration installed. 
​


<Frame>
  <img src="/images/1675373151.png" alt="" />

2. The messages would be sent from the contact owner or lead owner’s WhatsApp number, irrespective of the user who created the workflow.


<Frame>
  <img src="/images/1675378331.png" alt="" />
</Frame>


3. The messages will only get sent if the user has Eazybe installed and Zoho integrated.


<Frame>
  <img src="/images/1675379169.png" alt="" />
</Frame>


4. The phone number should contain the country code; otherwise, the message will be sent to the wrong number.


<Frame>
  <img src="/images/1675463143.png" alt="" />
</Frame>


5. The media file should not be more than 15mb, or the scheduled message will fail to send.
