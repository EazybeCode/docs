## Overview

As a super-admin in HubSpot, you can grant access and permissions to users for the Eazybe integration. This guide walks you through the process of enabling the necessary permissions to ensure smooth integration between Eazybe and HubSpot.

<Note>
Only HubSpot super-admins can modify user permissions. Contact your HubSpot administrator if you need permission changes.
</Note>

---

## Granting User Access

Follow these steps to grant users access to Eazybe through HubSpot:

<Steps>
  <Step title="Access HubSpot Settings">
    Log into your HubSpot account and navigate to the **Settings** page by clicking the gear icon in the top navigation bar.

    <img src="/images/1182407641.png" alt="HubSpot Settings" />
  </Step>

  <Step title="Navigate to Users & Teams">
    In the left sidebar, select **Users & Teams** to view all users in your HubSpot account.

    <img src="/images/1182409870.png" alt="Users and Teams Section" />
  </Step>

  <Step title="Edit User Permissions">
    Find the user you want to grant access to and click on the **Edit permissions** button next to their name.

    <img src="/images/1182411479.png" alt="Edit Permissions Button" />
  </Step>

  <Step title="Enable Required Permissions">
    Ensure that the following permissions are enabled for the user:
    - **File permissions** — Allows access to file management features
    - **App Marketplace permissions** — Enables integration with Eazybe

    <img src="/images/1182439947.png" alt="Permission Settings" />
  </Step>
</Steps>

<Check>
The user now has the necessary permissions to use Eazybe with HubSpot.
</Check>

## Required Permissions Explained

<CardGroup cols={2}>
  <Card title="File Permissions" icon="file">
    Enables users to manage files within HubSpot, which is essential for chat backup functionality and media synchronization.
  </Card>
  <Card title="App Marketplace Permissions" icon="store">
    Allows users to access and use third-party integrations like Eazybe from the HubSpot App Marketplace.
  </Card>
</CardGroup>

---

## Why These Permissions Matter

<Info>
These permissions are essential for ensuring smooth integration of Eazybe with HubSpot:
- **Chat Backup** — Syncing WhatsApp conversations to HubSpot contacts
- **CRM Synchronization** — Keeping contact data in sync between platforms
- **File Sharing** — Sharing media files between WhatsApp and HubSpot
- **Team Collaboration** — Enabling team members to access shared features
</Info>

## Best Practices

<AccordionGroup>
  <Accordion title="Grant Minimum Required Permissions">
    Only enable the permissions that users need for their specific role to maintain security best practices.
  </Accordion>

  <Accordion title="Review Permissions Regularly">
    Periodically audit user permissions to ensure they align with current team roles and responsibilities.
  </Accordion>

  <Accordion title="Document Permission Changes">
    Keep track of permission changes for compliance and troubleshooting purposes.
  </Accordion>
</AccordionGroup>
