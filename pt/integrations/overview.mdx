## CRM Integrations

Eazybe integrates with leading CRMs to sync contacts, backup conversations, and automate workflows.

### Supported CRMs

| CRM | Contact Sync | Chat Backup | Mini-CRM | Workflows | Dynamic Labels |
|-----|:------------:|:-----------:|:--------:|:---------:|:--------------:|
| HubSpot | ✅ | ✅ | ✅ | ✅ | ✅ |
| Zoho CRM | ✅ | ✅ | ✅ | ✅ | ✅ |
| Salesforce | ✅ | ✅ | ✅ | ✅ | ✅ |
| Pipedrive | ✅ | ✅ | ✅ | ✅ | - |
| LeadSquared | ✅ | ✅ | ✅ | ✅ | - |
| Bitrix24 | ✅ | ✅ | ✅ | - | - |
| Freshdesk | ✅ | ✅ | - | - | - |
| Google Sheets | ✅ | ✅ | - | - | - |

### Key Features

<CardGroup cols={2}>
  <Card title="Contact Sync" icon="address-book">
    Automatically match WhatsApp contacts with CRM records by phone number. Create new contacts directly from chats.
  </Card>
  <Card title="Chat Backup" icon="cloud-arrow-up">
    Save all WhatsApp conversations to CRM contact timelines. Never lose important customer communications.
  </Card>
  <Card title="Mini-CRM View" icon="sidebar">
    Access CRM data without leaving WhatsApp. View contact details, deals, and history in the sidebar.
  </Card>
  <Card title="Workflow Automation" icon="diagram-project">
    Trigger CRM workflows from WhatsApp events. Send automated messages when deals change stage.
  </Card>
  <Card title="Dynamic Labels" icon="tags">
    Sync Eazybe labels with CRM properties. Label a chat and update CRM fields automatically.
  </Card>
  <Card title="Team Management" icon="users">
    Control team access to integrations. Assign permissions and monitor sync status.
  </Card>
</CardGroup>

### Getting Started

<Steps>
  <Step title="Choose Your CRM">
    Select from our [supported integrations](#supported-crms)
  </Step>
  <Step title="Connect Account">
    Authorize Eazybe to access your CRM
  </Step>
  <Step title="Configure Sync">
    Set up contact matching and backup preferences
  </Step>
  <Step title="Start Using">
    Open WhatsApp Web and see CRM data in the sidebar
  </Step>
</Steps>
