## What Are Message Templates?

Message templates are pre-approved message formats required for sending outbound messages to customers outside the 24-hour conversation window. Every broadcast message must use an approved template.

## Template Components

| Component | Description | Required |
|-----------|-------------|----------|
| **Header** | Text, image, video, or document | Optional |
| **Body** | Main message content with variables | Required |
| **Footer** | Small text at bottom | Optional |
| **Buttons** | Call-to-action or quick reply | Optional |

## Variables

Use `{{1}}`, `{{2}}`, etc. to insert dynamic content:

```
Hi {{1}},

Your order #{{2}} has been confirmed!
Expected delivery: {{3}}

Thank you for shopping with us!
```
