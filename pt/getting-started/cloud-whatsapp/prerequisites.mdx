Antes de poder usar o EazyBe Cloud WhatsApp, certifique-se de atender aos seguintes requisitos:

<CardGroup cols={2}>
  <Card title="Extensao EazyBe" icon="puzzle-piece">
    A extensao EazyBe deve estar instalada e conectada ao seu WhatsApp Web
  </Card>
  <Card title="Plano Teams" icon="users">
    Voce precisa de uma assinatura do Plano Teams para acessar os recursos do Cloud
  </Card>
</CardGroup>

## Lista de Requisitos

<Steps>
  <Step title="Instalar Extensao EazyBe">
    Certifique-se de que a extensao EazyBe esteja instalada no seu navegador e conectada a sua conta do WhatsApp Web.
  </Step>

  <Step title="Assinar o Plano Teams">
    Mantenha uma assinatura do **Plano Teams** para aproveitar todos os beneficios do EazyBe Cloud WhatsApp.
  </Step>
</Steps>

<Info>
Depois de atender a ambos os requisitos, voce pode comecar a usar o Cloud WhatsApp para sincronizar conversas e agendar mensagens mesmo quando seu laptop estiver fechado.
</Info>
