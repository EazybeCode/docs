## Overview

Message templates are pre-written messages with placeholders for dynamic content. They're the foundation of efficient WhatsApp communication, enabling personalized messages at scale.

## Types of Templates

### Quick Reply Templates
For use with the Chrome Extension:
- Created and stored in Eazybe
- Can include attachments and voice notes
- Shared within your organization

### WABA Templates
For official WhatsApp Business API:
- Must be approved by Meta
- Required for outbound messaging
- Categories: Marketing, Utility, Authentication

## Creating Effective Templates

### Use Dynamic Variables
Replace static text with variables that get filled in automatically:

```
Hi {{name}},

Your order #{{order_id}} has been shipped!
Expected delivery: {{delivery_date}}

Track here: {{tracking_link}}
```

### Available Variables

| Variable | Description | Example |
|----------|-------------|---------|
| `{{name}}` | Contact's name | John |
| `{{company}}` | Company name | Acme Inc |
| `{{date}}` | Current date | Jan 11, 2026 |
| `{{time}}` | Current time | 2:30 PM |
| `{{custom}}` | Custom field | Any value |

## Best Practices

<AccordionGroup>
  <Accordion title="Keep it conversational">
    Write like a human, not a robot. Use natural language and appropriate emojis.
  </Accordion>
  
  <Accordion title="Include a clear CTA">
    Every template should have a clear next step - reply, click, call, etc.
  </Accordion>
  
  <Accordion title="Test before deploying">
    Send templates to yourself or colleagues first to catch any issues.
  </Accordion>
  
  <Accordion title="Update regularly">
    Review and refresh templates quarterly to keep them relevant.
  </Accordion>
</AccordionGroup>
