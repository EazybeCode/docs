This guide walks you through creating a custom HubSpot Analytics dashboard to track your WhatsApp messaging metrics and performance data. Build visual reports that help you monitor response times, message volumes, and team effectiveness.

## Creating Your Analytics Dashboard

<Steps>
  <Step title="Access Reporting">
    In your HubSpot account, click on **Reporting** in the main navigation menu.

    ![Click on Reporting](/images/hubspot-analytics-dashboard-1.png)
  </Step>
  <Step title="Open Reports">
    Click on **Reports** from the dropdown menu.

    ![Click on Reports](/images/hubspot-analytics-dashboard-2.png)
  </Step>
  <Step title="Create New Report">
    Click on **Create** button in the top right corner.

    ![Click on Create](/images/hubspot-analytics-dashboard-3.png)
  </Step>
  <Step title="Select Create Report">
    Click on **Create report** from the options.

    ![Click on Create report](/images/hubspot-analytics-dashboard-4.png)
  </Step>
  <Step title="Proceed to Builder">
    Click on **Next** to continue to the report builder.

    ![Click on Next](/images/hubspot-analytics-dashboard-5.png)
  </Step>
  <Step title="Enable Data Sources">
    On the Custom Report Builder screen, turn on **Add more data sources** to access additional metrics.

    ![Turn on Add more data sources](/images/hubspot-analytics-dashboard-6.png)
  </Step>
  <Step title="Continue to Data Selection">
    Click on **Next** to proceed to the data source selection.

    ![Click on Next](/images/hubspot-analytics-dashboard-7.png)
  </Step>
  <Step title="Search for Data Sources">
    Use the search bar to find the specific data sources you want to include in your dashboard.

    ![Search for data source](/images/hubspot-analytics-dashboard-8.png)
  </Step>
</Steps>

## Available WhatsApp Data Sources

You can choose from the following WhatsApp messaging data sources:

| Data Source                        | Description                               |
| ---------------------------------- | ----------------------------------------- |
| **Average Response Time**          | Average time taken to respond to messages |
| **Client Replied**                 | Number of client replies received         |
| **Current Label**                  | Current label assigned to conversation    |
| **Current Main Label**             | Main category label                       |
| **Current Sub Label**              | Subcategory label                         |
| **First Response Time**            | Time to first response                    |
| **Follow-ups**                     | Number of follow-up messages sent         |
| **Last AI Summary**                | Most recent AI-generated summary          |
| **Last Message sent by**           | Sender of the last message                |
| **Message Received**               | Total messages received                   |
| **Message sent**                   | Total messages sent                       |
| **Message you got**                | Count of incoming messages                |
| **Message you sent**               | Count of outgoing messages                |
| **Time Since Last Client Message** | Elapsed time since last client message    |
| **Total Messages**                 | Combined message count                    |

<Note>
  The availability of these data sources depends on your Eazybe integration configuration and the data synced from WhatsApp.
</Note>

## Building Your Dashboard

<Steps>
  <Step title="Drag Data to Axis">
    Click on your desired data source and drag it to the appropriate axis based on your reporting requirements. For example, drag **Total Messages** to visualize message volume.

    ![Drag data source to axis](/images/hubspot-analytics-dashboard-9.png)
  </Step>
  <Step title="Add More Metrics">
    Continue dragging and dropping additional data sources based on your dashboard requirements. Each metric can be configured for different visualization types.

    ![Drag and drop data sources](/images/hubspot-analytics-dashboard-10.png)
  </Step>
  <Step title="Save Your Report">
    Once you've configured all desired metrics and visualizations, click on **Save report** to finalize your analytics dashboard.

    ![Save the report](/images/hubspot-analytics-dashboard-11.png)
  </Step>
</Steps>

## Your Analytics Dashboard is Ready

Your custom HubSpot Analytics dashboard is now configured and ready to use. You can access this dashboard anytime to monitor your WhatsApp messaging performance and track key metrics that matter to your team.

<Tip>
  **Pro Tip**: Schedule regular dashboard reviews with your team to identify trends, optimize response times, and improve overall customer engagement strategies.
</Tip>
