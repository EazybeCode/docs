## The Solution

Eazybe automatically creates Pipedrive persons when new WhatsApp conversations start.

## Step-by-Step Setup

<Steps>
  <Step title="Enable Auto-Create">
    **Settings** → **Integrations** → **Pipedrive** → Toggle **Auto-create persons**
    
    <Frame>
      <img src="/images/pipedrive/auto-create-toggle.png" alt="Auto-create toggle" />
    </Frame>
  </Step>
  
  <Step title="Configure Mapping">
    | WhatsApp | Pipedrive Field |
    |----------|-----------------|
    | Phone | Phone |
    | Profile Name | Name |
    | - | Label = "WhatsApp Lead" |
    
    <Frame>
      <img src="/images/pipedrive/field-mapping.png" alt="Field mapping" />
    </Frame>
  </Step>
</Steps>
