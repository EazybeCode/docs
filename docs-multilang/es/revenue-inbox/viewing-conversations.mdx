## Opening the Team Inbox

<Steps>
  <Step title="Access Workspace">
    Go to **Eazybe Workspace** and log in with your credentials
  </Step>
  
  <Step title="Navigate to Team Inbox">
    Click **Team Inbox** in the left sidebar
  </Step>
  
  <Step title="Select Employee or Team">
    Choose to view:
    - **All Employees** - See every conversation
    - **Specific Employee** - Filter to one team member
    - **Specific Team** - Filter to a department
  </Step>
</Steps>

## Conversation View

Each conversation displays:

| Element | Description |
|---------|-------------|
| **Contact Name** | Customer's name (from WA or CRM) |
| **Phone Number** | Contact's phone number |
| **Last Message** | Preview of most recent message |
| **Time** | When the last message was sent |
| **Labels** | Applied tags (Hot Lead, VIP, etc.) |
| **Status** | Unreplied, Critical, Active, etc. |
| **Owner** | Assigned team member |

## Reading a Conversation

1. Click on any conversation row
2. Full chat history opens in the detail panel
3. See all messages with timestamps
4. View attachments (images, documents, voice notes)

## Quick Actions

From the conversation view:

<CardGroup cols={2}>
  <Card title="Assign" icon="user-plus">
    Assign the conversation to a team member
  </Card>
  <Card title="Add Label" icon="tag">
    Apply labels for organization
  </Card>
  <Card title="Add Note" icon="sticky-note">
    Add internal notes (not visible to customer)
  </Card>
  <Card title="Set Follow-up" icon="bell">
    Schedule a reminder for future action
  </Card>
</CardGroup>

## Search & Filter

### Search

Type in the search bar to find conversations by:
- Contact name
- Phone number
- Message content

### Filters

| Filter | Options |
|--------|---------|
| **Employee** | All, or select specific |
| **Team** | All, or select department |
| **Status** | Unreplied, Critical, Active, etc. |
| **Label** | Any applied label |
| **Date Range** | Filter by time period |

## Bulk Actions

Select multiple conversations to:
- Assign to a team member
- Apply labels
- Export to CSV
