## The Solution

Eazybe creates LeadSquared leads when new WhatsApp conversations start.

  <img src="/images/leadsquared/lead-capture-solution.gif" alt="Auto-create lead" />

## Setup

<Steps>
  <Step title="Enable Auto-Create">
    **Settings** → **LeadSquared** → Toggle **Auto-create leads**
    
    <Frame>
      <img src="/images/leadsquared/auto-create-toggle.png" alt="Auto-create" />
    </Frame>
  </Step>
  
  <Step title="Configure Mapping">
    | WhatsApp | LeadSquared Field |
    |----------|-------------------|
    | Phone | Mobile |
    | Name | First Name |
    | - | Source = "WhatsApp" |
  </Step>
</Steps>
