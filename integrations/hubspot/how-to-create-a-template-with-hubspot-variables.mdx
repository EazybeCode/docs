Q. What is the fallback value?
A. It is the default value, which gets overwritten at the place of the HubSpot property value if in case it's not available under the property. Used to make the personalisation and campaigns intact.

#### 
After the successful Approval, you can use the template for your HubSpot Workflows. 


#### 
Constraints:
1. Don't use the variables at the beginning and end of the Message Body.

2. Keep the number of variables and content size in consideration. Don't use a lot of variables with small content size.
3. If there is no contact owner associated with the contact on HubSpot, or the integrated HubSpot doesn't have access to it, then the fallback value will get placed with the corresponding variable.
4. The Template name should be unique. ( In case, when you delete the last template with the same name )
