Deseja enviar uma mensagem para vários contatos ao mesmo tempo. Criamos um recurso de mensagem de transmissão para você.


- Clique no botão transmitir mensagens na barra superior.


<Frame>
  <img src="/images/1112608994.png" alt="" />
</Frame>


2. Selecione a Resposta Rápida no menu suspenso que você deseja enviar como transmissão.


<Frame>
  <img src="/images/1112609104.png" alt="" />
</Frame>


3. Selecione o marcador para o qual deseja enviar a resposta rápida.


<Frame>
  <img src="/images/1112609231.png" alt="" />
</Frame>


4. Você também verá uma opção para Criar Nova Resposta Rápida ou pode enviar a partir da selecionada clicando no botão Enviar Mensagem.


<Frame>
  <img src="/images/1112609389.png" alt="" />
</Frame>


### 

Nota: Este recurso deve ser usado com precaução e de forma limitada. Não se esqueça de ler o aviso corretamente após clicar no botão Enviar mensagem.
