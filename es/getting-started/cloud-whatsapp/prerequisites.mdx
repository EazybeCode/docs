---

Antes de poder usar EazyBe Cloud WhatsApp, asegurate de cumplir con los siguientes requisitos:

<CardGroup cols={2}>
  <Card title="Extension EazyBe" icon="puzzle-piece">
    La extension EazyBe debe estar instalada y conectada a tu WhatsApp Web
  </Card>
  <Card title="Plan Teams" icon="users">
    Necesitas una suscripcion al Plan Teams para acceder a las funciones de Cloud
  </Card>
</CardGroup>

## Lista de Requisitos

<Steps>
  <Step title="Instalar Extension EazyBe">
    Asegurate de que la extension EazyBe este instalada en tu navegador y conectada a tu cuenta de WhatsApp Web.
  </Step>

  <Step title="Suscribirse al Plan Teams">
    Manten una suscripcion al **Plan Teams** para disfrutar de todos los beneficios de EazyBe Cloud WhatsApp.
  </Step>
</Steps>

<Info>
Una vez que cumplas ambos requisitos, podras comenzar a usar Cloud WhatsApp para sincronizar conversaciones y programar mensajes incluso cuando tu laptop este cerrada.
</Info>
