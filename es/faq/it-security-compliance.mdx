Eazybe es una extensión de Chrome que mejora WhatsApp Web con funciones de productividad e integración con CRM para equipos de ventas y soporte. Las capacidades clave incluyen mini-vistas de CRM, bandejas de entrada de equipo compartidas, etiquetas de contactos y sincronización de datos a Google Drive o sistemas CRM.

## Manejo y Almacenamiento de Datos

| Categoría | Detalles |
|-----------|----------|
| **Almacenamiento de Mensajes** | Eazybe no almacena mensajes. Todos los chats permanecen en WhatsApp. |
| **Datos de CRM** | Se muestran vía APIs; credenciales no almacenadas; OAuth preferido |
| **Copias de Seguridad** | Almacenadas en el Google Drive del usuario |
| **Archivos Adjuntos** | Medios almacenados en Google Drive cuando se habilita la copia de seguridad |
| **Almacenamiento en Servidor** | Los datos permanecen del lado del cliente o en los servicios de nube del usuario |

## Configuración Predeterminada de Respaldo de Chats

<Warning>
Cuando Team Inbox está habilitado, todos los chats de WhatsApp se respaldan por defecto a Google Drive.
</Warning>

Los usuarios pueden controlar la configuración de respaldo mediante:
- **Etiquetas "No Sincronizar"** para contactos específicos
- **Controles de administrador** para reglas de sincronización de toda la organización

## Conectividad de Cloud WhatsApp

Función opcional que permite mensajes programados/automatizados cuando el navegador está cerrado:

- No se almacenan datos de WhatsApp en los servidores de Eazybe
- Autenticación mediante código QR de WhatsApp
- Las sesiones se pueden desconectar en cualquier momento desde la extensión

## Permisos Requeridos

| Permiso | Propósito |
|---------|-----------|
| **Acceso a Pestañas de Chrome** | Superposición de UI para WhatsApp Web |
| **Acceso a CRM** | Integración OAuth/API con HubSpot, Zoho, Salesforce, etc. |
| **Google Drive** | Respaldo opcional y almacenamiento de Team Inbox |

## Requisitos de Configuración de TI

<Steps>
  <Step title="Requisitos del Navegador">
    Se requiere el navegador Google Chrome con inicio de sesión en WhatsApp Web en el mismo perfil de Chrome.
  </Step>
  <Step title="Permisos de Extensión">
    Habilitar los permisos de extensión de Chrome para Eazybe.
  </Step>
  <Step title="Opcional: Integración de CRM">
    Configurar claves API de CRM o credenciales OAuth si usa funciones de CRM.
  </Step>
  <Step title="Opcional: Google Drive">
    Otorgar acceso a Google Drive para administradores de workspace si usa funciones de respaldo.
  </Step>
  <Step title="Configuración de Firewall">
    Permitir HTTPS a `web.eazybe.com`, `api.eazybe.com` y sus endpoints de CRM.
  </Step>
</Steps>

<Tip>
Eazybe no requiere instalación de software en las máquinas de los usuarios - solo una extensión ligera de Chrome. Recomendamos usar cuentas de prueba para una evaluación segura.
</Tip>
