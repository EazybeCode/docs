## What is Mini-CRM View?

Mini-CRM View brings HubSpot's functionality directly into WhatsApp. Without leaving your conversation, you can:

- View and edit contact details
- Create and manage deals
- Add tasks and notes
- Log calls
- Create support tickets

## Accessing Mini-CRM

1. Open any WhatsApp chat
2. Look for the **HubSpot icon** (orange) in the Eazybe sidebar
3. Click to open the Mini-CRM panel

## Quick Actions

From any linked contact, you can:

| Action | Shortcut |
|--------|----------|
| Edit contact | Click property to edit |
| Create deal | + New Deal button |
| Add task | + Add Task button |
| Add note | + Add Note button |
| Log call | Phone icon |
| Create ticket | + New Ticket button |
| View in HubSpot | External link icon |

## Benefits

- **No tab switching** - Stay in WhatsApp while managing CRM
- **Real-time sync** - Changes reflect immediately in HubSpot
- **Context preservation** - See CRM data while chatting
- **Faster updates** - One-click actions for common tasks
