## Resumen

Esta guía proporciona instrucciones paso a paso sobre cómo vincular sus números API de WABA con usuarios específicos de HubSpot. El mapeo exitoso de usuarios garantiza una sincronización precisa de datos y una gestión eficiente del flujo de trabajo entre sus plataformas.

<Note>
Antes de continuar, asegúrese de tener **permisos de administrador** tanto en Eazybe como en HubSpot para administrar los mapeos de usuarios.
</Note>

---

## Guía Paso a Paso

<Steps>
  <Step title="Navegue a la Página de Integraciones">
    Vaya al panel de Eazybe y navegue a la sección **Integraciones**.

    ![Navegue a la página de integraciones](/images/waba-hubspot-1.png)

    <Tip>
    Puede acceder a la página de Integraciones desde el menú principal en la barra lateral izquierda.
    </Tip>
  </Step>

  <Step title="Acceda a la Configuración de Integración de HubSpot">
    Busque la tarjeta de integración de **HubSpot** y haga clic en ella para acceder a la configuración de integración.

    ![Acceda a la configuración de integración de HubSpot](/images/waba-hubspot-2.png)
  </Step>

  <Step title="Seleccione la Pestaña de Mapeo de Usuarios">
    En la configuración de integración de HubSpot, localice y haga clic en la pestaña **Mapeo de Usuarios**.

    ![Seleccione la pestaña de mapeo de usuarios](/images/waba-hubspot-3.png)

    <Info>
    Esta pestaña le permite asignar usuarios específicos de HubSpot para administrar sus números API de WABA.
    </Info>
  </Step>

  <Step title="Elija un Usuario de HubSpot para Mapear">
    Seleccione el **usuario de HubSpot** que desea mapear a un número API de WABA específico de la lista de usuarios disponibles.

    ![Elija un usuario de HubSpot para mapear](/images/waba-hubspot-4.png)
  </Step>

  <Step title="Seleccione el Usuario Específico de HubSpot">
    Elija el usuario específico de HubSpot del menú desplegable que desea asociar con su número API de WABA.

    ![Seleccione el usuario específico de HubSpot](/images/waba-hubspot-5.png)

    <Warning>
    Asegúrese de que el usuario seleccionado tenga los permisos necesarios en HubSpot para administrar las comunicaciones de la API.
    </Warning>
  </Step>

  <Step title="Verifique el Mapeo Exitoso del Usuario">
    Confirme el mapeo y verifique que el usuario ha sido vinculado exitosamente al número API de WABA.

    ![Verifique el mapeo exitoso del usuario](/images/waba-hubspot-6.png)

    <Tip>
    Puede modificar o eliminar los mapeos de usuarios en cualquier momento desde la pestaña Mapeo de Usuarios.
    </Tip>
  </Step>
</Steps>

## Solución de Problemas

<Accordion>
  <AccordionItem title="¿Usuario no aparece en la lista?">
    Asegúrese de que el usuario haya sido invitado a su portal de HubSpot y haya aceptado la invitación. Solo los usuarios activos de HubSpot con los permisos adecuados aparecerán en la lista de mapeo.
  </AccordionItem>
  <AccordionItem title="¿El mapeo no se guarda?">
    Verifique que tiene los permisos de administrador necesarios. Si el problema persiste, intente actualizar la página y volver a intentar el mapeo.
  </AccordionItem>
  <AccordionItem title="¿Puedo mapear múltiples números API a un usuario?">
    Sí, un solo usuario de HubSpot puede ser mapeado a múltiples números API de WABA. Esto es útil para los miembros del equipo que administran múltiples números de negocio.
  </AccordionItem>
</Accordion>
