## Visão Geral

Quando você tem muitos contatos conectados ao HubSpot, pode não querer que todas as conversas sincronizem automaticamente. O Eazybe fornece um sistema baseado em etiquetas que permite excluir seletivamente contatos específicos da sincronização de backup de chat.

<Note>
Este recurso é útil para excluir contatos pessoais, chats internos da equipe ou qualquer conversa que você não queira registrar no HubSpot.
</Note>

---

## Como Funciona

O Eazybe usa um **sistema de exclusão baseado em etiquetas**. Você cria uma etiqueta compartilhada, adiciona os contatos que deseja excluir e então desativa a sincronização para toda essa etiqueta. Esta abordagem facilita o gerenciamento de múltiplos contatos de uma só vez.

---

## Guia Passo a Passo

<Steps>
  <Step title="Criar uma Etiqueta Compartilhada">
    Primeiro, crie uma etiqueta compartilhada dentro da extensão Eazybe de WhatsApp. Adicione todos os contatos que você **não quer** que os backups de chat sincronizem com o HubSpot sob esta etiqueta.

    <img src="/images/1185335801.png" alt="Criar Etiqueta Compartilhada" />

    <Tip>
    Nomeie sua etiqueta com algo descritivo como "Sem Sincronização" ou "Excluir do Backup" para fácil identificação.
    </Tip>
  </Step>

  <Step title="Acessar Configurações de Backup e Sincronização">
    Clique no ícone de **Backup e Sincronização** na barra lateral direita da extensão do WhatsApp. Na janela resultante, selecione o botão **HubSpot** para acessar as configurações de sincronização específicas do HubSpot.
  </Step>

  <Step title="Desativar Sincronização para a Etiqueta">
    Localize a etiqueta contendo os contatos que você deseja excluir. Clique no botão **Não Sincronizar** ao lado da etiqueta para desativar a sincronização para todos os contatos nessa etiqueta.

    <img src="/images/1185336317.png" alt="Botão Não Sincronizar" />

    <Check>
    A sincronização de chat agora está desativada para todos os contatos nesta etiqueta!
    </Check>
  </Step>
</Steps>

---

## Reativar a Sincronização

Se você precisar restaurar a sincronização para contatos previamente excluídos:

<Steps>
  <Step title="Ativar Sincronização para a Etiqueta">
    Navegue de volta para as configurações de Backup e Sincronização, encontre sua etiqueta excluída e clique no botão **Sincronizar** para restaurar a sincronização.

    <img src="/images/1185339683.png" alt="Botão Sincronizar" />
  </Step>

  <Step title="Reativação Seletiva">
    Para ativar seletivamente a sincronização apenas para contatos específicos:

    1. Remova esses contatos da etiqueta excluída
    2. Mantenha outros contatos na etiqueta para manter sua exclusão
    3. Os contatos removidos retomarão automaticamente a sincronização
  </Step>
</Steps>

## Melhores Práticas

<AccordionGroup>
  <Accordion title="Use Nomes de Etiqueta Descritivos">
    Nomeie suas etiquetas de exclusão claramente (ex., "Sem Sincronização HubSpot", "Contatos Pessoais") para que os membros da equipe entendam seu propósito.
  </Accordion>

  <Accordion title="Revise as Exclusões Periodicamente">
    Revise regularmente seus contatos excluídos para garantir que você não está perdendo conversas de negócios importantes.
  </Accordion>

  <Accordion title="Documente sua Política de Exclusão">
    Se você faz parte de uma equipe, documente quais tipos de contatos devem ser excluídos para manter a consistência.
  </Accordion>
</AccordionGroup>
