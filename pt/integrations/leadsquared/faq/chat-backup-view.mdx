Suas conversas do WhatsApp são automaticamente salvas no LeadSquared através do EazyBe. Veja onde você pode encontrar esses backups de chat dentro do LeadSquared.

## Locais de Backup

Os backups de chats do WhatsApp com leads do LeadSquared estão acessíveis em dois locais principais:

### 1. Página de Leads

Os backups de chat aparecem na página dedicada de cada lead dentro do LeadSquared. Navegue até o perfil de qualquer lead para visualizar o histórico completo de conversas do WhatsApp.

![Backup de Chat na Página do Lead](/images/1188146542.png)

### 2. Página de Oportunidades do Lead

As conversas também são salvas na seção de Oportunidades do perfil de cada lead. Isso permite que você visualize o histórico de chat no contexto de oportunidades específicas.

![Backup de Chat na Página de Oportunidades](/images/1188146266.png)

<Info>
  Ambos os locais exibem os mesmos dados de conversa do WhatsApp, permitindo que você acesse o histórico de chat a partir da visualização mais relevante para seu fluxo de trabalho.
</Info>

<Tip>
  Use a Página de Leads para revisão geral de conversas e a Página de Oportunidades quando precisar consultar chats no contexto de um negócio ou oportunidade específica.
</Tip>
