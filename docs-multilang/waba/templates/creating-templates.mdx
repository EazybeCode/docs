**Creating WhatsApp templates in Eazybe makes it easy to send approved messages or quick replies to your contacts. The extension offers a guided process to save and reuse your frequently sent messages as templates.**

## Instructions

## Step 1: Access "Quick Reply" in the Chat Input Field

- Open the Eazybe extension on your WhatsApp web interface.
- In any chat window, locate the chat input section.
Click on the **"Quick Reply"** option below and within the input field to open quick reply actions.

<Frame>
  <img src="/images/1710847024.png" alt="" />
</Frame>

## Step 2: Switch On the "Create Template" Toggle

- In the quick reply menu, turn **ON** the toggle beside "Create Template."
This enables template creation mode, allowing you to set up new reusable message templates.

<Frame>
  <img src="/images/1710847407.png" alt="" />
</Frame>

## Step 3: Fill Template Details

- Enter required details for your template, including:

Template Name
- Language
- Category
- Template Type

If you want to attached any media select **Media** from **Template Type **drop-down and select the required media Image, Video or Document from **Media Type **drop-down
- If you don't want to attach media. Please add a **Header in text format only**.
- Add the message content as required.
- Add footer (Optional)
- You can add upto 2 Call-to-action button by linking to any url or phone number strictly in the following format **[https://www.example.com](https://www.example.com) format**
- Review the template for accuracy and completeness.

<Frame>
  <img src="/images/1726023464.png" alt="" />
</Frame>

## Step 4: Create Template

- Click on the **"Create"** button to save your new template.
Your template will now be available in the quick reply list for instant use in future chats.

<Frame>
  <img src="/images/1726058979.png" alt="" />
</Frame>

## Use Case

- Templates can include personalization placeholders (like customer names or invoice numbers).
- Saved templates are accessible in any chat via the quick reply menu.
- Use templates to speed up customer support, sales outreach, and broadcast messaging.
