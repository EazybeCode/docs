---
title: "AI Agent Setup"
description: "Learn how to create and configure your AI Knowledge Base Agent from start to finish"
---

This guide provides step-by-step instructions on how to create a new AI agent, define its purpose, and configure various aspects like tools, communication channels, and billing preferences.

<Steps>
    <Step>
        <h3>Access the Agent Gallery</h3>
        <p>Navigate to the AI Agents section in your dashboard to access the Agent Gallery. This is where you can view all available agent types and start creating your custom agent.</p>
    </Step>
    <Step>
        <h3>Start Custom Agent Creation</h3>
        <p>Click the <strong>Create Agent</strong> button to begin the custom agent creation process. You'll be guided through each step of setting up your AI agent.</p>
    </Step>
    <Step>
        <h3>Describe Your Agent's Purpose and Instructions</h3>
        <p>You'll see a text box where you can describe what you want your agent to do. Be clear and specific about your requirements.</p>

        <p>You can write your description in various formats:</p>
        <ul>
            <li><strong>A single line:</strong> "Reply to my WhatsApp customer inquiries about pricing"</li>
            <li><strong>Bullet points:</strong> "Greet the customer, ask what they need, share pricing, collect their email"</li>
            <li><strong>Detailed instructions:</strong> "When a customer asks about pricing, share our rate card. If they ask for a discount, tell them to talk to the sales team."</li>
        </ul>

        <Note>You can write in any language you're comfortable with — Hindi, English, Hinglish, Spanish, or more. The better your description, the smarter your agent will be.</Note>
    </Step>
    <Step>
        <h3>Click on Create Agent to Initiate Agent Build</h3>
        <p>Click <strong>Create Agent</strong> when you're ready. The system will read your description and build a complete agent for you.</p>

        <p>This includes:</p>
        <ul>
            <li><strong>Role</strong> — who the agent is (e.g., "You are a sales assistant for XYZ company")</li>
            <li><strong>Goal</strong> — what it should achieve (e.g., "Convert inquiries into booked demos")</li>
            <li><strong>Flow</strong> — step-by-step conversation flow (e.g., "First greet, then ask about their need, then share pricing")</li>
            <li><strong>Tools</strong> — what actions the agent can take (e.g., "Search knowledge base, schedule follow-up")</li>
        </ul>

        <p>You don't need to write any of this yourself — it's all generated from your description. If the system needs more clarity, it will ask before finalizing.</p>
    </Step>
    <Step>
        <h3>Engage with Memory Match (Optional)</h3>
        <p>Memory Match helps enhance your agent's capabilities by learning from existing data. This optional step allows your agent to better understand context and provide more accurate responses.</p>
    </Step>
    <Step>
        <h3>Review Agent Building Progress and Provide Additional Agent Details</h3>
        <p>Monitor your agent's building progress and provide any additional details requested by the system. This ensures your agent is properly configured before deployment.</p>
    </Step>
    <Step>
        <h3>Click on "Add" to Submit</h3>
        <p>Click on <strong>Add</strong> to submit your agent configuration once you're satisfied with the setup.</p>
    </Step>
    <Step>
        <h3>Submit Agent Access Details for Workflows</h3>
        <p>Configure the necessary access details for workflows. This step ensures your agent can interact with other systems and tools as needed.</p>
    </Step>
    <Step>
        <h3>Proceed to Agent Configuration for More Customizations</h3>
        <p>Proceed to the Agent Configuration section for more customizations. Here you can fine-tune your agent's behavior, responses, and capabilities.</p>
    </Step>
    <Step>
        <h3>Navigate to the Test Tab and Input Test Query to Test the Agent</h3>
        <p>Navigate to the <strong>Test</strong> tab and input a test query to verify your agent is working correctly. This is an important step to ensure your agent responds as expected before going live.</p>
    </Step>
    <Step>
        <h3>View Agent Conversations</h3>
        <p>Monitor your agent's conversations to understand how it's interacting with users. This helps you identify areas for improvement.</p>
    </Step>
    <Step>
        <h3>Go to Learning Tab and Click on "Add Learning" to Add Learnings</h3>
        <p>Go to the <strong>Learning</strong> tab and click on <strong>Add Learning</strong> to teach your agent new information. This is where you can continuously improve your agent's knowledge base.</p>

        <p>You can add:</p>
        <ul>
            <li><strong>Documents</strong> — upload PDFs, brochures, price lists, FAQs, product catalogs</li>
            <li><strong>Website URLs</strong> — paste your website link, help docs, product pages. The system reads and learns from them automatically</li>
            <li><strong>WhatsApp Chat Sync</strong> — train the agent on your existing WhatsApp conversations. It learns your tone, style, and how you talk to customers</li>
        </ul>

        <Tip>The more knowledge you give your agent, the better it will answer customer questions accurately instead of making things up.</Tip>
    </Step>
    <Step>
        <h3>Click on "Tools" and Review Configured Agent Tools</h3>
        <p>Click on <strong>Tools</strong> to review your configured agent tools. Each tool represents a capability your agent can use.</p>

        <p>Available tools include:</p>
        <ul>
            <li><strong>Search Knowledge Base</strong> — agent can look up answers from your uploaded docs and website</li>
            <li><strong>Send WhatsApp Messages</strong> — agent can reply to customers</li>
            <li><strong>Schedule Follow-ups</strong> — agent can set reminders and send follow-up messages automatically</li>
            <li><strong>Transfer to Human</strong> — agent hands over to your team when needed, and stops replying</li>
            <li><strong>Create/Update CRM Contacts</strong> — agent can save customer details to your CRM</li>
            <li><strong>Google Sheets, Calendar, Gmail</strong> — connect external apps for tasks like booking appointments or logging data</li>
        </ul>
    </Step>
    <Step>
        <h3>Examine Enabled and Available Tools</h3>
        <p>Review which tools are currently enabled and see additional available tools you can activate. Turn on the tools you need and configure them (e.g., set who gets notified on handoff).</p>
    </Step>
    <Step>
        <h3>Access WhatsApp Integration and Configure Agent's WhatsApp Number</h3>
        <p>Access the WhatsApp Integration section to configure your agent's WhatsApp number. This allows your agent to interact with customers on WhatsApp.</p>

        <p>You can choose any WhatsApp Business API or Coexistence number. Your WhatsApp Business app keeps working exactly as before — nothing changes. Both you and the agent can reply to the same customer at the same time, and your customers won't know they're talking to an AI.</p>

        <Note>Connection takes about 2 minutes. Once connected, every incoming message on that number goes through your agent first.</Note>
    </Step>
    <Step>
        <h3>Navigate to Billing Preferences and Review LLM Billing Options Based on Requirement</h3>
        <p>Navigate to <strong>Billing Preferences</strong> and review LLM billing options based on your requirements. You can see exactly what each AI action costs and control your spending.</p>

        <p>Available options:</p>
        <ul>
            <li><strong>Light LLM</strong> — faster responses, lower cost. Best for simple Q&A, FAQs, and straightforward conversations</li>
            <li><strong>Heavy LLM</strong> — smarter responses, higher cost. Best for complex conversations, negotiations, and tasks that need deeper understanding</li>
        </ul>

        <Tip>You can switch between Light and Heavy anytime. The billing dashboard shows you cost per message, total spend, and usage breakdown.</Tip>
    </Step>
    <Step>
        <h3>Select LLM Model Type</h3>
        <p>Choose the appropriate LLM model type based on your use case and budget. Consider the complexity of conversations your agent will handle.</p>
    </Step>
    <Step>
        <h3>Save Agent Configuration and Model</h3>
        <p>Click <strong>Save</strong> to store your agent configuration and model settings. This ensures all your settings are preserved.</p>
    </Step>
    <Step>
        <h3>Click on Publish to Finish the Setup and Activate the AI Agent</h3>
        <p>Click on <strong>Publish</strong> to finish the setup and activate your AI Agent. Your agent is now ready to assist customers!</p>

        <Warning>Make sure to test your agent thoroughly before publishing to ensure it responds as expected.</Warning>
    </Step>
</Steps>
