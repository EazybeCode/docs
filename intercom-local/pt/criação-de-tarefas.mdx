Seção de Tarefas

Você sabia que EazyBe oferece o privilégio de executar tarefas HubSpot no próprio WhatsApp.

Você está se perguntando como?

Vamos descobrir.

- Selecione o contato para o qual deseja criar uma tarefa.
- Abra o chat do contato e clique no ícone HubSpot. O segundo botão na barra superior o levará para a seção Tarefas.
- Você pode criar uma tarefa conforme indicado abaixo.

  

4. Existe uma disposição para selecionar a prioridade das tarefas porque as mais importantes vêm primeiro.

  

5. Você pode criar várias tarefas clicando no ícone de adição.
