## Performance Metrics

Monitor your team's WhatsApp performance with detailed analytics.

### Key Metrics

| Metric | Description | Target |
|--------|-------------|--------|
| **Response Time** | Time to first reply | < 5 min |
| **Resolution Time** | Time to close conversation | < 24 hrs |
| **Messages Sent** | Total outbound messages | - |
| **Conversations Handled** | Unique chats managed | - |
| **Customer Satisfaction** | Feedback ratings | > 4.5/5 |

### Team Leaderboard

View performance by team member:
- Messages sent
- Conversations resolved
- Average response time
- Customer ratings

### Time-Based Analysis

Filter metrics by:
- Today
- Last 7 days
- Last 30 days
- Custom date range

### Exporting Reports

1. Go to Dashboard → Analytics
2. Select date range
3. Click **Export**
4. Download CSV or PDF

<Info>
Schedule automated weekly reports to your email in Settings → Reports.
</Info>
