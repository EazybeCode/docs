Você pode criar um ticket diretamente no HubSpot.

Quer saber, como? Vamos descobrir.


- Vá até o contato para o qual deseja criar um ticket.


- Clique no ícone HubSpot. O último botão na barra superior o levará para a seção Ingressos.
