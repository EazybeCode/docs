Este guia explica como criar e usar Modelos de Mensagens para enviar uma transmissão (ou iniciar qualquer conversa) com WABA.

## O que é um Modelo de Mensagem?

Um Modelo de Mensagem é um formato de mensagem pré-aprovado que permite enviar notificações de saída ou iniciar conversas com seus clientes. Cada modelo deve ser enviado à Meta para aprovação para prevenir spam e garantir comunicação de alta qualidade.

<Info>
Modelos de Mensagens são necessários para iniciar conversas fora da janela de mensagens de 24 horas ou enviar campanhas de transmissão.
</Info>

## Como Criar um Modelo de Mensagem

<Steps>
  <Step title="Acessar a Página de Modelos">
    Navegue até a seção **Modelos** no seu painel do Eazybe.

    ![Página de Modelos](/images/1589150371.png)
  </Step>

  <Step title="Criar Novo Modelo">
    Selecione o botão **Criar Modelo** para começar.

    ![Botão Criar Modelo](/images/1589152450.png)
  </Step>

  <Step title="Preencher os Detalhes do Modelo">
    Complete os seguintes campos:

    ![Configuração do Modelo](/images/1589154268.png)

    - **Nome do Modelo:** Um nome único em minúsculas com underscores (ex., `oferta_natal_v2`)
    - **Categoria:** Selecione o tipo de mensagem:
      - **Marketing** - Mensagens promocionais e ofertas
      - **Utilidade** - Mensagens transacionais como atualizações de pedidos
      - **Autenticação** - OTPs e códigos de verificação
    - **Idioma:** Escolha o idioma do modelo

    ### Componentes Opcionais

    - **Cabeçalho:** Adicione texto, imagem, vídeo ou documento no topo
    - **Corpo:** Este é o texto principal da sua mensagem
    - **Rodapé:** Linha curta de texto na parte inferior
    - **Botões:** Adicione até dois botões - Respostas Rápidas (ex., "Sim," "Não") ou Chamadas para Ação (ex., "Visitar Site," "Ligar")

    ![Opções de Botões](/images/1589158457.png)

    <Tip>
    Use variáveis como `{{1}}`, `{{2}}` no corpo do seu modelo para personalizar mensagens com nomes de clientes, números de pedidos ou outro conteúdo dinâmico.
    </Tip>
  </Step>

  <Step title="Enviar para Aprovação">
    Clique em **Enviar** para enviar o modelo à Meta para revisão.

    ![Enviar Modelo](/images/1589159135.png)

    <Warning>
    Revise seu modelo cuidadosamente antes de enviar. Certifique-se de que não há erros de ortografia ou violações de políticas que possam causar rejeição.
    </Warning>
  </Step>
</Steps>

## Verificar Status do Modelo

Após o envio, seu modelo exibirá um dos seguintes status:

![Lista de Status dos Modelos](/images/1589160140.png)

![Exemplos de Status dos Modelos](/images/1589160753.png)

<AccordionGroup>
  <Accordion title="Pendente">
    O modelo está em revisão pela Meta. Isso pode levar de alguns minutos até 24 horas.
  </Accordion>
  <Accordion title="Aprovado">
    Seu modelo está pronto para usar em transmissões e conversas com clientes.
  </Accordion>
  <Accordion title="Rejeitado">
    O modelo não atendeu às políticas da Meta. Revise o feedback fornecido, faça as alterações necessárias e reenvie.
  </Accordion>
</AccordionGroup>
