## Genel Bakis

Bu kilavuz, yoneticilere Zoho CRM'de profil duzeyinde API erisimini etkinlestirme talimatlari saglar. API erisimini etkinlestirmek, Eazybe'nin Zoho CRM verilerinizle iletisim kurabilmesi icin gereklidir.

## Bu Kilavuzu Kimler Kullanmali

<CardGroup cols={3}>
  <Card title="CRM Yoneticileri" icon="user-shield">
    Kullanici izinlerini yoneten Zoho CRM yoneticileri
  </Card>
  <Card title="BT/Entegrasyon Ekipleri" icon="network-wired">
    Sistem entegrasyonlarini kuran ekipler
  </Card>
  <Card title="Geliştiriciler" icon="code">
    API tabanli entegrasyonlari yapilandiran gelistiriciler
  </Card>
</CardGroup>

<Note>
  Zoho CRM'de yalnizca yonetici erisimi olan kullanicilar profil duzeyindeki guvenlik ayarlarini degistirebilir.
</Note>

## On Kosullar

Devam etmeden once, asagidakilere sahip oldugunuzdan emin olun:

- Zoho CRM'e **yonetici erisimi**
- API cagrilari yapan kullaniciya hangi **profilin** atandigini bilmek (Standart, Satis veya ozel profiller)

## Dogrulama

API erisimini etkinlestirdikten sonra, yapilandirmayi dogrulayin:

<AccordionGroup>
  <Accordion title="Profil Durumunu Kontrol Edin">
    Profil, **API Erisimi** durumunu **ACIK** olarak gostermelidir.
  </Accordion>

  <Accordion title="Kimlik Dogrulamayi Test Edin">
    Kullanici kimlik dogrulamasi API kimlik bilgileri veya OAuth tokenlari ile basarili olmalidir.
  </Accordion>

  <Accordion title="API Cagrilarini Dogrulayin">
    API cagrilari artik izin hatalari dondurmemelidir.
  </Accordion>
</AccordionGroup>

## Sorun Giderme

<Warning>
  API cagrilari erisimi etkinlestirdikten sonra hala basarisiz oluyorsa, asagidaki yaygin sorunlari kontrol edin:
</Warning>

| Sorun | Cozum |
|-------|-------|
| Yanlis profil guncellendi | API kullanicisina atanan dogru profili duzenleyip duzenlemediginizi dogrulayin |
| Oturum yenilenmedi | Kullanicidan cikis yapmasini ve tekrar giris yapmasini isteyin |
| IP kisitlamalari | IP tabanli erisim kisitlamalarinin API cagrilarini engelleyip engellemedigini kontrol edin |
| OAuth kapsami | OAuth tokeninin gerekli kapsamlara sahip olduğunu dogrulayin |

## Guvenlik En Iyi Uygulamalari

<Tip>
  API erisimini yonetirken bu guvenlik onerilerini izleyin:
</Tip>

- **Secici sekilde etkinlestirin** - API erisimini yalnizca bunu gerektiren profiller icin etkinlestirin
- **Genis erişimden kacinin** - API erisimini tum profiller icin gereksiz yere etkinlestirmeyin
- **Duzenli denetimler** - Hangi profillerin API erisiminin etkinlestirildigini periyodik olarak gozden gecirin
- **Kullanilmadiginda iptal edin** - Entegrasyonlar artik kullanilmadiginda API erisimini devre disi birakin
