## Chat Backup

Save all WhatsApp conversations to Bitrix24 contact records.

### Enabling Backup

1. Go to **Settings** > **Bitrix24** > **Chat Backup**
2. Enable **Auto-Backup**
3. Choose backup frequency

### What Gets Backed Up

- Text messages
- Timestamps
- Sender information
- Message direction
