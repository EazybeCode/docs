Depois que um rótulo/funil for criado, você verá muitas opções.

- Clique no nome do rótulo/funil e os contatos do mesmo serão abertos.
- Clique no menu suspenso conforme indicado abaixo.
- Você verá uma lista de opções conforme fornecido abaixo.
- Adicionar contatos ao rótulo/funil - Você pode adicionar os contatos.
- Marcar todos como lidos - Marca todos os chats do rótulo/funil como lidos.
- Editar rótulo/funil - Você pode editar as propriedades usando este recurso.
- Upload via excel - Para fazer upload dos contatos usando uma planilha excel.
- Baixar contatos - O arquivo será baixado em formato .xls e conterá os contatos da etiqueta/funil.
- Mensagens de difusão - Você pode criar uma mensagem de difusão para enviar a todos os contatos do marcador. Quer saber como criar uma mensagem de difusão, clique neste link para saber.
- No lado direito da barra intermediária, você também pode ver um menu suspenso onde pode alternar os sub-rótulos/estágios do rótulo/funil criado, respectivamente.
