## Overview

Monitor how your chatbots are performing and identify opportunities for improvement.

## Key Metrics

| Metric | Description | Target |
|--------|-------------|--------|
| **Containment Rate** | % handled without human | > 70% |
| **Handoff Rate** | % transferred to human | < 30% |
| **Completion Rate** | % reaching flow end | > 80% |
| **Avg. Resolution Time** | Time to resolve | < 2 min |
| **Customer Satisfaction** | Post-chat rating | > 4/5 |

## Dashboard

Access analytics at **Automation** → **Agents** → **Analytics**

### Overview Tab
- Total conversations handled
- Containment vs. handoff ratio
- Peak usage times
- Trend over time

### Flow Analysis
- Drop-off points in flow
- Most used paths
- Least used options
- Error rates by node

### Customer Feedback
- Satisfaction ratings
- Feedback comments
- Common complaints

## Identifying Issues

### High Drop-off

If customers abandon the flow:
- Simplify options
- Shorten messages
- Add more handoff points

### High Handoff Rate

If too many transfer to humans:
- Analyze handoff reasons
- Add missing FAQ answers
- Improve question understanding

### Low Satisfaction

If ratings are poor:
- Review conversation transcripts
- Identify confusing messages
- Test flow personally
