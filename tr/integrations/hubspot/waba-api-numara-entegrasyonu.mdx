## Özet

Bu rehber, WABA API numaralarınızı belirli HubSpot kullanıcılarına nasıl bağlayacağınıza dair adım adım talimatlar sağlar. Kullanıcıların başarıyla eşleştirilmesi, platformlarınız arasında doğru veri senkronizasyonunu ve verimli iş akışı yönetimini sağlar.

<Note>
Devam etmeden önce, hem Eazybe hem de HubSpot'ta kullanıcı eşleştirmelerini yönetmek için **Yönetici izinlerine** sahip olduğunuzdan emin olun.
</Note>

---

## Adım Adım Rehber

<Steps>
  <Step title="Entegrasyonlar Sayfasına Gitin">
    Eazybe paneline gidin ve **Entegrasyonlar** bölümüne gidin.

    ![Entegrasyonlar sayfasına gidin](/images/waba-hubspot-1.png)

    <Tip>
    Entegrasyonlar sayfasına sol kenar çubuğundaki ana menüden ulaşabilirsiniz.
    </Tip>
  </Step>

  <Step title="HubSpot Entegrasyon Ayarlarına Erişin">
    **HubSpot** entegrasyon kartını bulun ve entegrasyon ayarlarına erişmek için üzerine tıklayın.

    ![HubSpot entegrasyon ayarlarına erişin](/images/waba-hubspot-2.png)
  </Step>

  <Step title="Kullanıcı Eşleştirme Sekmesini Seçin">
    HubSpot entegrasyon ayarlarında, **Kullanıcı Eşleştirme** sekmesini bulun ve tıklayın.

    ![Kullanıcı eşleştirme sekmesini seçin](/images/waba-hubspot-3.png)

    <Info>
    Bu sekme, WABA API numaralarınızı yönetmek için belirli HubSpot kullanıcıları atamanıza olanak tanır.
    </Info>
  </Step>

  <Step title="Eşleştirmek İçin Bir HubSpot Kullanıcısı Seçin">
    Mevcut kullanıcı listesinden belirli bir WABA API numarasına eşleştirmek istediğiniz **HubSpot kullanıcısını** seçin.

    ![Eşleştirmek için bir HubSpot kullanıcısı seçin](/images/waba-hubspot-4.png)
  </Step>

  <Step title="Belirli HubSpot Kullanıcısını Seçin">
    WABA API numaranızla ilişkilendirmek istediğiniz belirli HubSpot kullanıcısını açılır menüden seçin.

    ![Belirli HubSpot kullanıcısını seçin](/images/waba-hubspot-5.png)

    <Warning>
    Seçilen kullanıcının API iletişimlerini yönetmek için HubSpot'ta gerekli izinlere sahip olduğundan emin olun.
    </Warning>
  </Step>

  <Step title="Başarılı Kullanıcı Eşleştirmesini Doğrulayın">
    Eşleştirmeyi onaylayın ve kullanıcının WABA API numarasına başarıyla bağlandığını doğrulayın.

    ![Başarılı kullanıcı eşleştirmesini doğrulayın](/images/waba-hubspot-6.png)

    <Tip>
    Kullanıcı Eşleştirme sekmesinden dilediğiniz zaman kullanıcı eşleştirmelerini değiştirebilir veya kaldırabilirsiniz.
    </Tip>
  </Step>
</Steps>

## Sorun Giderme

<Accordion>
  <AccordionItem title="Kullanıcı listede görünmüyor mu?">
    Kullanıcının HubSpot portalinize davet edildiğinden ve davetiye kabul ettiğinden emin olun. Yalnızca uygun izinlere sahip aktif HubSpot kullanıcıları eşleştirme listesinde görünür.
  </AccordionItem>
  <AccordionItem title="Eşleştirme kaydedilmiyor mu?">
    Gerekli yönetici izinlerine sahip olduğunuzu kontrol edin. Sorun devam ederse, sayfayı yenilemeyi deneyin ve eşleştirmeyi tekrar yapmayı deneyin.
  </AccordionItem>
  <AccordionItem title="Birden fazla API numarasını bir kullanıcıya eşleştirebilir miyim?">
    Evet, tek bir HubSpot kullanıcısı birden fazla WABA API numarasına eşleştirilebilir. Bu, birden fazla iş numarasını yöneten ekip üyeleri için kullanışlıdır.
  </AccordionItem>
</Accordion>
