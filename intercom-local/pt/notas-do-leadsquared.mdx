Você tem uma conversa importante que não quer perder de vista, continuando, ou uma discussão que parou no meio do caminho antes de chegar a uma conclusão? Não se preocupe! Com a seção Notas na visualização EazyBe LeadSquared Mini-CRM, você pode criar notas de acompanhamento e salvar mensagens importantes para referência futura.

Mas você pode estar se perguntando, como?

Vamos descobrir.

- 
#### Selecione o contato
Abra o contato para o qual deseja criar um acompanhamento. Certifique-se de que o contato exista no LeadSquared.

- 
#### Acesse a seção Notas
Clique no ícone LeadSquared. A segunda seção na barra superior o levará para a **seção Notas**.

- 
#### Adicione uma nova nota
Clique no botão **Adicionar nota** para adicionar notas para o respectivo lead.

  

- 
#### Insira os detalhes da nota
Digite a nota ou descrição que deseja armazenar para a conversa e clique em **Salvar**. A nota será salva junto com o horário da criação.

  

- 
#### Editar ou Excluir Notas
Você pode **Editar** ou **Excluir** uma nota, clicar nos três pontos na parte superior e selecionar a ação desejada.

- 
#### Adicionar Várias Notas
Você pode adicionar notas adicionais clicando no botão **Adicionar Nota** e isso criará novas notas para esse lead.

  

**Nota: Criar Notas para Oportunidades LeadSquared**

Você também pode criar **Notas** para qualquer Oportunidade LeadSquared de forma semelhante às tarefas:

- Clique na respectiva oportunidade para a qual deseja adicionar uma nota.
- Siga os mesmos passos para adicionar uma nota clicando no botão **Adicionar Nota**.
- Uma vez que a nota for adicionada, ela será salva naquela oportunidade específica para referência futura.
