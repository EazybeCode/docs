Sección de Tareas

¿Sabes? EazyBe te brinda el privilegio de realizar tareas de HubSpot en el propio WhatsApp.

¿Te preguntas cómo?

Vamos a averiguar.

- Seleccione el contacto para el que desea crear una tarea.
- Abre el chat del contacto y haz clic en el ícono de HubSpot. El segundo botón en la barra superior te llevará a la sección Tareas.
- Puede crear una tarea como se indica a continuación.

  

4. Existe una disposición para seleccionar la prioridad de las tareas porque las más importantes son las primeras.

  

5. Puede crear varias tareas haciendo clic en el icono más.
