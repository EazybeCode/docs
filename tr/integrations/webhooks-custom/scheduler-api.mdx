Zamanlayici API, kimlik dogrulamasi yapilmis organizasyon kullanicilarinin WhatsApp mesajlarini programatik olarak zamanlamasina olanak tanir. API, kullanici organizasyon uyeligini dogrular, gerekli mesaj detaylarini kontrol eder ve izin sorunlarini yonetir.

<Warning>
Bu API'yi spam veya toplu mesajlasma icin kullanmayin. Kotuye kullanim hesap yasaklanmasina neden olabilir.
</Warning>

## Endpoint

```
POST https://api.eazybe.com/v2/scheduler/public-scheduler
```

## Kimlik Dogrulama

EazyBe Workspace'inizden **Organizasyon Kimlik Dogrulama Tokeni** almaniz gerekiyor.

<Steps>
  <Step title="EazyBe Workspace'e Erisin">
    [workspace.eazybe.com](https://workspace.eazybe.com) adresine gidin
  </Step>

  <Step title="Organizasyon Sayfasina Gidin">
    Bir yoneticinin Organizasyon sayfasina erismesi gerekiyor.
  </Step>

  <Step title="Kimlik Dogrulama Tokenini Kopyalayin">
    Organizasyon Kimlik Dogrulama Tokeni bu sayfada gosterilecektir. API kullanimi icin kopyalayin.

    ![Organizasyon Kimlik Dogrulama Tokeni](/images/1294609565.png)
  </Step>
</Steps>

## Istek Payload'u

API asagidaki alanlari gerektirir:

| Alan | Tip | Aciklama |
|------|-----|----------|
| `from` | string | Gondericinin organizasyonel email adresi |
| `to` | string | Alicinin telefon numarasi (uluslararasi format) |
| `message` | string | Gonderilecek metin icerigi |
| `name` | string | Bildirimler icin alicinin adi |
| `time` | string | Zamanlanmis teslimat zamani (ISO 8601 formati) |

### Ornek Payload

```json
{
  "from": "hello@eazybe.com",
  "to": "919876543210",
  "message": "Hello World",
  "name": "Vats",
  "time": "2024-03-01T17:25:00.000Z"
}
```

## Kod Ornegi (Node.js)

```javascript
const axios = require('axios');

const config = {
  method: 'post',
  url: 'https://api.eazybe.com/v2/scheduler/public-scheduler',
  headers: {
    'Authorization': 'Bearer YOUR_ORGANIZATION_AUTH_TOKEN',
    'Content-Type': 'application/json'
  },
  data: {
    from: 'hello@eazybe.com',
    to: '919876543210',
    message: 'Hello World',
    name: 'Vats',
    time: '2024-03-01T17:25:00.000Z'
  }
};

axios(config)
  .then(response => console.log(response.data))
  .catch(error => console.error(error));
```
