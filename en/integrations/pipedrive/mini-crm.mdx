## Overview

The Pipedrive Mini CRM allows you to manage your sales activities directly from WhatsApp without switching applications. Create contacts, add notes, manage tasks, and create deals based on your WhatsApp conversations.

<Note>
Make sure you have [connected your Pipedrive account](/en/integrations/pipedrive/account-connection) before using these features.
</Note>

## Creating Contacts

<Steps>
  <Step title="Open a WhatsApp Chat">
    Select the chat for which you want to create a contact in Pipedrive.

    <!-- TODO: Add screenshot of WhatsApp chat selected -->
    <img src="/images/TODO-pipedrive-whatsapp-chat.png" alt="WhatsApp Chat Selected" />
  </Step>

  <Step title="Open Mini CRM">
    Click on the Pipedrive logo to open the Mini CRM.
  </Step>

  <Step title="Create Contact">
    Under Mini CRM, click on **Create Contact**.

    <!-- TODO: Add screenshot of Create Contact button/option -->
    <img src="/images/TODO-pipedrive-create-contact.png" alt="Create Contact Button" />

    The contact will be created based on the information from the open WhatsApp chat.
  </Step>
</Steps>

## Quick Reference

| Feature | Description |
|---------|-------------|
| **Create Contact** | Add WhatsApp contacts to Pipedrive |
| **Notes** | Add notes to contacts from conversations |
| **Tasks** | Create and manage tasks for follow-ups |
| **Deals** | Create deals based on sales conversations |
| **Contact Redirect** | Open contact's full Pipedrive page |
| **Chat Backup** | View WhatsApp chat history in contacts |
