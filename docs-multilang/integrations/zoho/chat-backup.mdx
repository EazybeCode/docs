## Chat Backup

Save all WhatsApp conversations to Zoho CRM contact records.

### Enabling Backup

1. Go to **Settings** > **Zoho CRM** > **Chat Backup**
2. Enable **Auto-Backup**
3. Choose backup frequency

### What Gets Backed Up

- Text messages
- Timestamps
- Sender information
- Message direction
