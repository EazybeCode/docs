## Genel Bakış

Eazybe'yi Odoo'ya bağlamadan önce, Odoo instance'ınızdan bir API anahtarı oluşturmanız gerekir. Bu anahtar, Eazybe'nin Odoo verilerinize güvenli bir şekilde erişmesine izin verecektir.

## Odoo API Anahtarı Oluşturma

<Steps>
  <Step title="Odoo Ayarlarını Açın">
    Odoo instance'ınıza giriş yapın ve üst menüden **Ayarlar**'a tıklayın.
    <img src="/images/odoo-04.png" alt="Odoo Ayarları" />
  </Step>

  <Step title="Kullanıcılar ve Şirketler'e Gidin">
    Ayarlar menüsünde, **Kullanıcılar ve Şirketler**'e ve ardından **Kullanıcılar**'a tıklayın.
  </Step>

  <Step title="Kullanıcınızı Seçin">
    Listeden kullanıcı hesabınızı bulun ve tıklayın.
    <img src="/images/odoo-05.png" alt="Kullanıcı Seç" />
  </Step>

  <Step title="Güvenlik Ayarlarına Erişin">
    Kullanıcı detayları sayfasında **Güvenlik** sekmesine tıklayın.
    <img src="/images/odoo-06.png" alt="Güvenlik Sekmesi" />
  </Step>

  <Step title="API Anahtarı Oluşturun">
    **API Anahtarları** bölümünde, **API Anahtarı Ekle**'ye tıklayın.
    <img src="/images/odoo-07.png" alt="API Anahtarı Ekle" />

    <Note>
    Daha sonra tanıyabilmeniz için API anahtarınıza "Eazybe Entegrasyonu" gibi tanınabilir bir ad verin.
    </Note>

    Oluşturmak için **Oluştur**'a tıklayın.
  </Step>

  <Step title="API Anahtarınızı Kopyalayın">
    <Check>
    API anahtarınız görüntülenecektir. Tekrar göremeyeceğiniz için hemen kopyalayın.
    </Check>
    <Warning>
    API anahtarınızı güvende tutun ve yetkisiz kullanıcılarla paylaşmayın.
    </Warning>
  </Step>
</Steps>

---

## Bağlantı İçin Gerekenler

API anahtarınıza sahip olduktan sonra, şunlara da ihtiyacınız olacak:

| Öğe | Açıklama |
|------|----------|
| **Odoo Instance URL'si** | Şirketinizin Odoo URL'si (ör. `https://sirketiniz.odoo.com`) |
| **API Anahtarı** | Az önce oluşturduğunuz anahtar |
