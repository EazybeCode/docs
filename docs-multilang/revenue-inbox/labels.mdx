## Labels

Labels help you categorize and organize WhatsApp conversations for better tracking and filtering.

### Creating Labels

<Steps>
  <Step title="Click Add Label">
    Click the **Add Label** button in the sidebar
    
    ![Add Label Button](/images/1113048889.png)
  </Step>
  
  <Step title="Choose Label Type">
    Select **Label** (for categorization) or **Funnel** (for stages)
    
    ![Label Type Selection](/images/1113049106.png)
  </Step>
  
  <Step title="Set Visibility">
    Choose visibility:
    - **Personal** - Only visible to you
    - **Shared** - Visible to your team/organization
    
    ![Visibility Options](/images/1113049302.png)
  </Step>
  
  <Step title="Add Sub-Labels">
    Create sub-labels for more granular organization
    
    ![Sub-Labels](/images/1113049518.png)
  </Step>
  
  <Step title="Assign Contacts">
    Select contacts to add to this label
    
    ![Assign Contacts](/images/1113049922.png)
  </Step>
</Steps>

### Applying Labels to Chats

1. Open a conversation in WhatsApp Web
2. Click the **Label** icon in the Eazybe sidebar
3. Select the label to apply
4. The chat will now appear under that label in Team Inbox

<Info>
Labels sync with your CRM when [Dynamic Labels](/integrations/hubspot/dynamic-labels) is enabled.
</Info>
