Tasks Section

Do you know, EazyBe provides you with the privilege to HubSpot Tasks in WhatsApp itself.

Are you wondering how?

Let’s find out.


- Select the contact for which you want to create a task for.


- Open the Contact’s chat and click on the HubSpot icon. The second button in the top bar will take you to the Tasks section.
