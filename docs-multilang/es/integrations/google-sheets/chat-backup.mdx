## Chat Backup

Save all WhatsApp conversations to Google Sheets contact records.

### Enabling Backup

1. Go to **Settings** > **Google Sheets** > **Chat Backup**
2. Enable **Auto-Backup**
3. Choose backup frequency

### What Gets Backed Up

- Text messages
- Timestamps
- Sender information
- Message direction
